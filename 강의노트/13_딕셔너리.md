## 1️⃣ **딕셔너리란 무엇인가?**

딕셔너리는 **키(key)와 값(value)의 쌍**으로 데이터를 저장합니다. 마치 실제 사전에서 단어(키)로 뜻(값)을 찾듯이 사용합니다.

```python
# 리스트 방식 (비효율적)
names = ["철수", "영희", "민수"]
phones = ["010-1111-1111", "010-2222-2222", "010-3333-3333"]
# 철수의 번호를 찾으려면?
index = names.index("철수")
print(phones[index])

# 딕셔너리 방식 (효율적!)
phonebook = {
    "철수": "010-1111-1111",
    "영희": "010-2222-2222",
    "민수": "010-3333-3333"
}
print(phonebook["철수"])  # 바로 찾기!
```

```
딕셔너리의 구조

   키(Key)    값(Value)
   ┌─────┐   ┌──────────────┐
   │철수 │ : │010-1111-1111 │
   ├─────┤   ├──────────────┤
   │영희 │ : │010-2222-2222 │
   ├─────┤   ├──────────────┤
   │민수 │ : │010-3333-3333 │
   └─────┘   └──────────────┘
```

## 2️⃣ **딕셔너리 생성하기**

딕셔너리는 중괄호 `{}`로 만들며, 키와 값은 콜론(`:`)으로 구분합니다.

```python
# 빈 딕셔너리
empty = {}

# 문자열 키
student = {
    "이름": "김철수",
    "나이": 20,
    "학과": "컴퓨터공학"
}

# 숫자 키도 가능
scores = {
    1: 85,
    2: 90,
    3: 78
}

print(student)  # {'이름': '김철수', '나이': 20, '학과': '컴퓨터공학'}
```

### **예제 1: 카페 메뉴판**

카페 메뉴를 딕셔너리로 관리하는 프로그램입니다.

```python
# 카페 메뉴 관리 프로그램
print("☕" + "=" * 38 + "☕")
print("   카페 메뉴판")
print("☕" + "=" * 38 + "☕")

# 메뉴 딕셔너리 (메뉴명: 가격)

    "아메리카노": 4500,
    "카페라떼": 5000,
    "카푸치노": 5500,

print("\n📋 메뉴 목록:")

for drink in menu:
    print(f"{drink}: {menu[drink]:,}원")

    print(f"합계: {total:,}원")
    print("-" * 40)
else:
    print(f"'{order}'는 메뉴에 없습니다.")
```

## 3️⃣ **딕셔너리 값 접근하기**

### **대괄호로 접근**

```python
student = {
    "이름": "김철수",
    "나이": 20,
    "학과": "컴퓨터공학"
}

print(student["이름"])  # 김철수
print(student["나이"])  # 20

# print(student["학번"])  # 오류! 키가 없으면 에러
```

### **get() 메서드 (안전한 접근)**

```python
# get(키) - 키가 없으면 None 반환
print(student.get("학과"))  # 컴퓨터공학
print(student.get("학번"))  # None (오류 안 남!)

# get(키, 기본값) - 키가 없으면 기본값 반환
print(student.get("학번", "정보없음"))  # 정보없음
```

## 4️⃣ **딕셔너리 수정하기**

딕셔너리는 변경 가능(mutable)합니다.

### **값 추가 및 수정**

```python
# 값 수정
person = {
    "이름": "홍길동",
    "나이": 25
}

person["나이"] = 26  # 수정
print(person)  # {'이름': '홍길동', '나이': 26}

# 새 키-값 추가
person["직업"] = "개발자"
print(person)  # {'이름': '홍길동', '나이': 26, '직업': '개발자'}
```

### **값 삭제**

```python
# del 키워드
student = {
    "이름": "김철수",
    "나이": 20,
    "학과": "컴퓨터공학"
}

del student["학과"]
print(student)  # {'이름': '김철수', '나이': 20}

# pop(키) - 값을 반환하며 제거
age = student.pop("나이")
print(age)      # 20
print(student)  # {'이름': '김철수'}
```

### **예제 2: 연락처 관리**

연락처를 추가, 수정, 삭제하는 프로그램입니다.

```python
# 연락처 관리 프로그램
print("📞" + "=" * 38 + "📞")
print("   연락처 관리")
print("📞" + "=" * 38 + "📞")

contacts = {}

while True:
    print("\n" + "=" * 40)
    print("1. 연락처 추가")
    print("2. 연락처 검색")
    print("3. 연락처 수정")
    print("4. 연락처 삭제")
    print("5. 전체 보기")
    print("6. 종료")
    print("=" * 40)
  
    choice = input("\n선택: ")
  
    if choice == "1":
        name = input("\n이름: ")
        phone = input("전화번호: ")
        contacts[name] = phone
        print(f"✓ {name} 추가됨")
  
    elif choice == "2":
        name = input("\n검색할 이름: ")
        if name in contacts:
            print(f"{name}: {contacts[name]}")
        else:
            print(f"'{name}'이(가) 없습니다.")
  
    elif choice == "3":
        name = input("\n수정할 이름: ")
        if name in contacts:
            print(f"현재 번호: {contacts[name]}")
            new_phone = input("새 번호: ")
            contacts[name] = new_phone
            print(f"✓ {name} 수정됨")
        else:
            print(f"'{name}'이(가) 없습니다.")
  
    elif choice == "4":
        name = input("\n삭제할 이름: ")
        if name in contacts:
            del contacts[name]
            print(f"✓ {name} 삭제됨")
        else:
            print(f"'{name}'이(가) 없습니다.")
  
    elif choice == "5":
        print("\n📇 전체 연락처")
        print("-" * 40)
        if len(contacts) == 0:
            print("연락처가 비어있습니다.")
        else:
            for name in contacts:
                print(f"{name}: {contacts[name]}")
        print("-" * 40)
        print(f"총 {len(contacts)}명")
  
    elif choice == "6":
        print("\n프로그램을 종료합니다.")
        break
  
    else:
        print("잘못된 선택입니다.")
```

## 5️⃣ **딕셔너리 메서드**

### **keys(), values(), items()**

```python
menu = {
    "아메리카노": 4500,
    "카페라떼": 5000,
    "카푸치노": 5500
}

# keys() - 모든 키
print(menu.keys())  
# dict_keys(['아메리카노', '카페라떼', '카푸치노'])

# values() - 모든 값
print(menu.values())  
# dict_values([4500, 5000, 5500])

# items() - 모든 키-값 쌍
print(menu.items())  
# dict_items([('아메리카노', 4500), ('카페라떼', 5000), ('카푸치노', 5500)])
```

### **반복문과 함께 사용**

```python
menu = {
    "아메리카노": 4500,
    "카페라떼": 5000,
    "카푸치노": 5500
}

# 키만 반복
for drink in menu:
    print(drink)

# 키와 값 모두 반복
for drink, price in menu.items():
    print(f"{drink}: {price}원")

# 값만 반복
for price in menu.values():
    print(f"{price}원")
```

### **예제 3: 재고 관리**

상품 재고를 관리하는 프로그램입니다.

```python
# 재고 관리 프로그램
print("📦" + "=" * 38 + "📦")
print("   재고 관리 시스템")
print("📦" + "=" * 38 + "📦")

# 상품 재고 (상품명: 수량)
inventory = {
    "노트북": 15,
    "마우스": 50,
    "키보드": 30,
    "모니터": 20,
    "헤드셋": 25
}

while True:
    print("\n" + "=" * 40)
    print("1. 재고 확인")
    print("2. 입고")
    print("3. 출고")
    print("4. 전체 현황")
    print("5. 종료")
    print("=" * 40)
  
    choice = input("\n선택: ")
  
    if choice == "1":
        product = input("\n상품명: ")
        quantity = inventory.get(product)
  
        if quantity is not None:
            print(f"\n{product} 재고: {quantity}개")
            if quantity < 10:
                print("⚠️ 재고 부족!")
        else:
            print(f"'{product}'는 등록되지 않은 상품입니다.")
  
    elif choice == "2":
        product = input("\n상품명: ")
        amount = int(input("입고 수량: "))
  
        if product in inventory:
            inventory[product] = inventory[product] + amount
        else:
            inventory[product] = amount
  
        print(f"✓ {product} {amount}개 입고 완료")
        print(f"현재 재고: {inventory[product]}개")
  
    elif choice == "3":
        product = input("\n상품명: ")
  
        if product not in inventory:
            print(f"'{product}'는 등록되지 않은 상품입니다.")
            continue
  
        amount = int(input("출고 수량: "))
  
        if inventory[product] >= amount:
            inventory[product] = inventory[product] - amount
            print(f"✓ {product} {amount}개 출고 완료")
            print(f"남은 재고: {inventory[product]}개")
        else:
            print(f"✗ 재고 부족! (현재: {inventory[product]}개)")
  
    elif choice == "4":
        print("\n📊 전체 재고 현황")
        print("-" * 40)
  
        total_items = 0
        low_stock = []
  
        for product, quantity in inventory.items():
            status = "⚠️" if quantity < 10 else "✓"
            print(f"{status} {product}: {quantity}개")
            total_items = total_items + quantity
      
            if quantity < 10:
                low_stock.append(product)
  
        print("-" * 40)
        print(f"총 상품 종류: {len(inventory)}개")
        print(f"총 재고 수량: {total_items}개")
  
        if len(low_stock) > 0:
            print(f"\n⚠️ 재고 부족 상품 ({len(low_stock)}개):")
            for product in low_stock:
                print(f"  • {product}")
  
    elif choice == "5":
        print("\n프로그램을 종료합니다.")
        break
  
    else:
        print("잘못된 선택입니다.")
```

### **예제 4: 학생 성적 관리**

학생별 과목 성적을 관리하는 프로그램입니다.

```python
# 학생 성적 관리 프로그램
print("📚" + "=" * 38 + "📚")
print("   학생 성적 관리 시스템")
print("📚" + "=" * 38 + "📚")

# 학생 정보 (이름: {과목: 점수})
students = {}

student_count = int(input("\n학생 수: "))

for i in range(student_count):
    print(f"\n[{i+1}번 학생]")
    name = input("이름: ")
    korean = int(input("국어: "))
    english = int(input("영어: "))
    math = int(input("수학: "))
  
    students[name] = {
        "국어": korean,
        "영어": english,
        "수학": math
    }

# 성적표 출력
print("\n" + "=" * 40)
print("전체 성적표")
print("=" * 40)

for name, scores in students.items():
    print(f"\n📝 {name}")
    print("-" * 40)
  
    total = 0
    for subject, score in scores.items():
        print(f"{subject}: {score}점")
        total = total + score
  
    average = total / len(scores)
  
    # 등급 계산
    if average >= 90:
        grade = "A"
    elif average >= 80:
        grade = "B"
    elif average >= 70:
        grade = "C"
    elif average >= 60:
        grade = "D"
    else:
        grade = "F"
  
    print("-" * 40)
    print(f"총점: {total}점")
    print(f"평균: {average:.1f}점")
    print(f"등급: {grade}")

print("\n" + "=" * 40)
```

## 7️⃣ **딕셔너리 컴프리헨션**

딕셔너리를 간결하게 만드는 방법입니다.

```python
# 기본 방법
squares = {}
for i in range(1, 6):
    squares[i] = i ** 2
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# 딕셔너리 컴프리헨션
squares = {i: i**2 for i in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

### **조건이 있는 컴프리헨션**

```python
# 짝수만
even_squares = {i: i**2 for i in range(1, 11) if i % 2 == 0}
print(even_squares)  # {2: 4, 4: 16, 6: 36, 8: 64, 10: 100}

# 값 변환
prices = {"사과": 1000, "바나나": 1500, "오렌지": 2000}
discounted = {item: price * 0.9 for item, price in prices.items()}
print(discounted)
# {'사과': 900.0, '바나나': 1350.0, '오렌지': 1800.0}
```

### **예제 5: 환율 계산**

여러 통화의 환율을 계산하는 프로그램입니다.

```python
# 환율 계산 프로그램
print("💱" + "=" * 38 + "💱")
print("   환율 계산기")
print("💱" + "=" * 38 + "💱")

# 기준 환율 (1달러 기준)
exchange_rates = {
    "USD": 1.0,
    "EUR": 0.85,
    "JPY": 110.0,
    "CNY": 6.5,
    "GBP": 0.73
}

amount_usd = float(input("\n달러 금액 ($): "))

# 딕셔너리 컴프리헨션으로 환산
converted = {
    currency: amount_usd * rate
    for currency, rate in exchange_rates.items()
}

# 결과 출력
print("\n" + "=" * 40)
print("환전 결과")
print("=" * 40)

currency_names = {

    "EUR": "유로",
    "JPY": "엔",
    "CNY": "위안",
    "GBP": "파운드"
}

for currency, amount in converted.items():
    name = currency_names[currency]
    print(f"{currency} ({name}): {amount:,.2f}")

print("=" * 40)
```

## 📝 **핵심 개념 정리**

딕셔너리는 중괄호 `{}`로 만들며, 키와 값은 콜론(`:`)으로 연결합니다. 키로 값을 빠르게 찾을 수 있어 효율적입니다.

대괄호 `[]`나 `get()` 메서드로 값에 접근합니다. `get()`은 키가 없을 때 오류 대신 None이나 기본값을 반환하여 안전합니다.

`keys()`, `values()`, `items()` 메서드로 딕셔너리의 모든 키, 값, 키-값 쌍을 얻을 수 있습니다. 반복문과 함께 사용하면 모든 항목을 처리할 수 있습니다.

딕셔너리 컴프리헨션은 `{키: 값 for 항목 in 시퀀스}` 형태로 간결하게 딕셔너리를 만드는 방법입니다. 중첩 딕셔너리로 복잡한 구조의 데이터를 체계적으로 관리할 수 있습니다.

문장에서 각 단어가 몇 번 나오는지 세는 프로그램을 작성하세요.

```python
# 힌트
text = "python is great python is easy python is fun"
words = text.split()

word_count = {}
for word in words:
    if word in word_count:
        word_count[word] = word_count[word] + 1
    else:
        word_count[word] = 1

# 결과 출력
```

## ✅ **퀴즈**

### **[초급] 1번**

딕셔너리를 올바르게 생성한 것은?

```python
1. my_dict = {"name": "철수", "age": 20}
2. my_dict = ["name": "철수", "age": 20]
3. my_dict = ("name": "철수", "age": 20)
4. my_dict = {"name" = "철수", "age" = 20}
```

### **[중급] 2번**

다음 코드의 실행 결과는?

```python
person = {"이름": "홍길동", "나이": 25}
print(person["이름"])
```

1. 홍길동
2. 이름
3. 25
4. 오류 발생

### **[중급] 3번**

다음 코드 실행 후 딕셔너리는?

```python
menu = {"커피": 4000}
menu["차"] = 3000
print(menu)
```

1. {"커피": 4000}
2. {"차": 3000}
3. {"커피": 4000, "차": 3000}
4. 오류 발생

### **[고급] 4번**

다음 코드의 실행 결과는?

```python
data = {"a": 1, "b": 2, "c": 3}
total = 0
for value in data.values():
    total = total + value
print(total)
```

1. 3
2. 6
3. abc
4. 오류 발생

### **[고급] 5번**

딕셔너리 컴프리헨션으로 1-5의 제곱을 만들기

```python
1. {i: i**2 for i in range(1, 6)}
2. [i: i**2 for i in range(1, 6)]
3. {i**2 for i in range(1, 6)}
4. {i, i**2 for i in range(1, 6)}
```

## 🔑 **퀴즈 정답 및 해설**

**1번 정답: 1**
딕셔너리는 중괄호 `{}`로 만들고, 키와 값은 콜론(`:`)으로 구분합니다.

**2번 정답: 1**
딕셔너리에서 `person["이름"]`은 "이름" 키에 해당하는 값인 "홍길동"을 반환합니다.

**3번 정답: 3**
`menu["차"] = 3000`은 딕셔너리에 새로운 키-값 쌍을 추가하므로, `{"커피": 4000, "차": 3000}`이 됩니다.

**4번 정답: 2**
`data.values()`는 모든 값 (1, 2, 3)을 반환하고, 이를 모두 더하면 1 + 2 + 3 = 6입니다.

**5번 정답: 1**
딕셔너리 컴프리헨션은 `{키: 값 for 항목 in 시퀀스}` 형태로 작성합니다.
