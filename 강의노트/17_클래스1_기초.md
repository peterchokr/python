## 1️⃣ **클래스란 무엇인가?**

지금까지 우리는 함수를 만들어 코드를 재사용했습니다. 하지만 프로그램이 복잡해지면 함수만으로는 부족합니다. 예를 들어, 게임에서 캐릭터를 만든다고 생각해봅시다.

```python
# 함수만 사용한 방식 (불편함!)
player_name = "용사"
player_health = 100
player_level = 1

def attack():
    print(f"{player_name}이(가) 공격!")

def heal():
    global player_health
    player_health += 20
```

캐릭터가 2명, 3명으로 늘어나면 어떻게 될까요? 변수가 엄청나게 많아지고 관리하기 힘들어집니다. 이럴 때 **클래스(Class)** 가 필요합니다!

### **클래스는 설계도다**

클래스는 마치 붕어빵 틀과 같습니다. 틀로 여러 개의 붕어빵을 만들 수 있듯이, 클래스로 여러 개의 객체를 만들 수 있습니다.

```
클래스와 객체의 관계

┌─────────────────┐
│   클래스        │  ← 설계도 (붕어빵 틀)
│   (Class)       │
└────────┬────────┘
         │ 생성
         ▼
┌─────────────────┐
│   객체 1        │  ← 실제 제품 (붕어빵1)
│   (Object)      │
└─────────────────┘
┌─────────────────┐
│   객체 2        │  ← 실제 제품 (붕어빵2)
│   (Object)      │
└─────────────────┘
┌─────────────────┐
│   객체 3        │  ← 실제 제품 (붕어빵3)
│   (Object)      │
└─────────────────┘
```

### **실생활 비유**

- **자동차 설계도** = 클래스
- **내 자동차, 네 자동차** = 객체
- **스마트폰 모델** = 클래스
- **내 아이폰, 네 아이폰** = 객체
- **사람** = 클래스
- **홍길동, 김철수** = 객체

### **클래스의 구성 요소**

클래스는 크게 두 가지로 구성됩니다:

**1. 속성(Attribute)**: 데이터, 특징

- 자동차 → 색상, 속도, 브랜드
- 사람 → 이름, 나이, 키

**2. 메서드(Method)**: 동작, 기능

- 자동차 → 출발하다(), 정지하다(), 가속하다()
- 사람 → 먹다(), 자다(), 걷다()

### **방법 2: __init__ 생성자 (권장)**

`__init__`은 객체를 만들 때 자동으로 실행되는 특별한 메서드입니다.

```python
class Dog:
    def __init__(self, name, age, breed):
        """생성자: 객체를 초기화"""
        self.name = name
        self.age = age
        self.breed = breed

# 객체 생성할 때 값 전달
my_dog = Dog("멍멍이", 3, "진돗개")
your_dog = Dog("바둑이", 5, "시바견")

# 사용
print(f"{my_dog.name}는 {my_dog.age}살 {my_dog.breed}")
print(f"{your_dog.name}는 {your_dog.age}살 {your_dog.breed}")
```

**실행 결과:**

```
멍멍이는 3살 진돗개
바둑이는 5살 시바견
```

### **self가 뭔가요?**

`self`는 "나 자신"을 의미합니다. 객체 자기 자신을 가리키는 변수입니다.

```python
class Dog:
    def __init__(self, name, age):
        self.name = name  # 나(이 객체)의 이름
        self.age = age    # 나(이 객체)의 나이

dog1 = Dog("멍멍이", 3)
# self는 dog1을 가리킴
# dog1.name = "멍멍이"
# dog1.age = 3

dog2 = Dog("바둑이", 5)
# self는 dog2를 가리킴
# dog2.name = "바둑이"
# dog2.age = 5
```

```
self의 역할

Dog("멍멍이", 3) 호출 시:
                  ┌──────────────┐
self ───────────► │   dog1 객체  │
                  ├──────────────┤
                  │ name: "멍멍이"│
                  │ age: 3       │
                  └──────────────┘

Dog("바둑이", 5) 호출 시:
                  ┌──────────────┐
self ───────────► │   dog2 객체  │
                  ├──────────────┤
                  │ name: "바둑이"│
                  │ age: 5       │
                  └──────────────┘
```

💡 **핵심**: `self`는 메서드를 호출한 객체 자신을 의미합니다!

## 4️⃣ **메서드 추가하기**

이제 클래스에 기능(메서드)을 추가해봅시다.

### **기본 메서드**

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
  
    def bark(self):
        """짖기"""
        print(f"{self.name}: 멍멍!")
  
    def info(self):
        """정보 출력"""
        print(f"이름: {self.name}, 나이: {self.age}살")

# 사용
my_dog = Dog("멍멍이", 3)
my_dog.bark()   # 멍멍이: 멍멍!
my_dog.info()   # 이름: 멍멍이, 나이: 3살
```

### **매개변수가 있는 메서드**

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        self.energy = 100
  
    def play(self, minutes):
        """놀기"""
        self.energy -= minutes
        print(f"{self.name}가(이) {minutes}분 놀았습니다.")
        print(f"남은 에너지: {self.energy}")
  
    def sleep(self, hours):
        """자기"""
        recovered = hours * 10
        self.energy += recovered
        if self.energy > 100:
            self.energy = 100
        print(f"{self.name}가(이) {hours}시간 잤습니다.")
        print(f"현재 에너지: {self.energy}")

# 사용
dog = Dog("멍멍이", 3)
dog.play(30)    # 에너지 30 감소
dog.sleep(5)    # 에너지 50 회복
```

## 5️⃣ **실전 예제: 학생 클래스**

학생 관리 시스템을 만들어봅시다.

```python
class Student:
    """학생 클래스"""
  
    def __init__(self, name, student_id, grade):
        """생성자"""
        self.name = name
        self.student_id = student_id
        self.grade = grade
        self.scores = []  # 성적 리스트
  
    def add_score(self, subject, score):
        """성적 추가"""
        self.scores.append({
            "subject": subject,
            "score": score
        })
        print(f"✓ {subject}: {score}점 추가됨")
  
    def get_average(self):
        """평균 계산"""
        if len(self.scores) == 0:
            return 0
  
        total = sum(s["score"] for s in self.scores)
        return total / len(self.scores)
  
    def show_info(self):
        """정보 출력"""
        print("\n" + "=" * 40)
        print(f"📚 학생 정보")
        print("=" * 40)
        print(f"이름: {self.name}")
        print(f"학번: {self.student_id}")
        print(f"학년: {self.grade}학년")
        print("\n📊 성적:")
  
        if len(self.scores) == 0:
            print("  등록된 성적이 없습니다.")
        else:
            for s in self.scores:
                print(f"  {s['subject']}: {s['score']}점")
      
            avg = self.get_average()
            print(f"\n평균: {avg:.1f}점")
  
        print("=" * 40)

# 학생 생성
student = Student("김철수", "2024001", 1)

# 성적 추가
student.add_score("국어", 85)
student.add_score("영어", 90)
student.add_score("수학", 78)

# 정보 출력
student.show_info()
```

========================================

학년: 1학년

📊 성적:
  국어: 85점

## 6️⃣ **실전 예제: 은행 계좌 클래스**

입출금이 가능한 은행 계좌를 만들어봅시다.

```python
class BankAccount:
    """은행 계좌 클래스"""
  
    def __init__(self, owner, balance=0):
        """
        생성자
        owner: 계좌 소유자
        balance: 초기 잔액 (기본값 0)
        """
        self.owner = owner
        self.balance = balance
        self.transactions = []  # 거래 내역
  
        # 계좌 개설 기록
        self.transactions.append(f"계좌 개설 (초기 잔액: {balance:,}원)")
  
    def deposit(self, amount):
        """입금"""
        if amount <= 0:
            print("❌ 입금액은 0보다 커야 합니다.")
            return False
  
        self.balance += amount
        self.transactions.append(f"입금: +{amount:,}원")
        print(f"✓ {amount:,}원 입금 완료")
        print(f"잔액: {self.balance:,}원")
        return True
  
    def withdraw(self, amount):
        """출금"""
        if amount <= 0:
            print("❌ 출금액은 0보다 커야 합니다.")
            return False
  
        if amount > self.balance:
            print(f"❌ 잔액이 부족합니다. (잔액: {self.balance:,}원)")
            return False
  
        self.balance -= amount
        self.transactions.append(f"출금: -{amount:,}원")
        print(f"✓ {amount:,}원 출금 완료")
        print(f"잔액: {self.balance:,}원")
        return True
  
    def get_balance(self):
        """잔액 조회"""
        return self.balance
  
    def show_transactions(self):
        """거래 내역 출력"""
        print("\n" + "=" * 40)
        print(f"💳 {self.owner}님의 거래 내역")
        print("=" * 40)
  
        for i, transaction in enumerate(self.transactions, 1):
            print(f"{i}. {transaction}")
  
        print("-" * 40)
        print(f"현재 잔액: {self.balance:,}원")
        print("=" * 40)

# 계좌 생성
account = BankAccount("홍길동", 10000)

print("\n💰 은행 계좌 시스템")
print()

# 입금
account.deposit(50000)
print()

# 출금
account.withdraw(20000)
print()

# 출금 실패 (잔액 부족)
account.withdraw(100000)
print()

# 거래 내역 확인
account.show_transactions()
```

**실행 결과:**

```
💰 은행 계좌 시스템

✓ 50,000원 입금 완료
잔액: 60,000원

✓ 20,000원 출금 완료
잔액: 40,000원

❌ 잔액이 부족합니다. (잔액: 40,000원)

========================================
💳 홍길동님의 거래 내역
========================================
1. 계좌 개설 (초기 잔액: 10,000원)
2. 입금: +50,000원
3. 출금: -20,000원
----------------------------------------
현재 잔액: 40,000원
========================================
```

## 7️⃣ **클래스 변수 vs 인스턴스 변수**

변수에는 두 종류가 있습니다.

### **인스턴스 변수**: 객체마다 다른 값

```python
class Dog:
    def __init__(self, name):
        self.name = name  # 인스턴스 변수

dog1 = Dog("멍멍이")
dog2 = Dog("바둑이")

print(dog1.name)  # 멍멍이
print(dog2.name)  # 바둑이
```

### **클래스 변수**: 모든 객체가 공유하는 값

```python
class Dog:
    species = "Canis familiaris"  # 클래스 변수 (모든 개가 공유)
  
    def __init__(self, name):
        self.name = name  # 인스턴스 변수 (개마다 다름)

dog1 = Dog("멍멍이")
dog2 = Dog("바둑이")

print(dog1.species)  # Canis familiaris
print(dog2.species)  # Canis familiaris (같은 값)

print(dog1.name)  # 멍멍이
print(dog2.name)  # 바둑이 (다른 값)
```

### **실전 예제: 게임 캐릭터**

```python
class Character:
    # 클래스 변수
    total_characters = 0  # 생성된 캐릭터 수
  
    def __init__(self, name, job):
        # 인스턴스 변수
        self.name = name
        self.job = job
        self.level = 1
        self.hp = 100
  
        # 캐릭터 생성시 카운트 증가
        Character.total_characters += 1
  
    def info(self):
        print(f"{self.name} ({self.job}) - Lv.{self.level}")
  
    @classmethod
    def get_total(cls):
        """전체 캐릭터 수 반환"""
        return cls.total_characters

# 캐릭터 생성
char1 = Character("용사", "전사")
char2 = Character("마법사", "마법사")
char3 = Character("궁수", "궁수")

# 정보 출력
char1.info()
char2.info()
char3.info()

print(f"\n총 {Character.get_total()}명의 캐릭터가 생성되었습니다.")
```

**실행 결과:**

```
용사 (전사) - Lv.1
마법사 (마법사) - Lv.1
궁수 (궁수) - Lv.1

총 3명의 캐릭터가 생성되었습니다.
```

## 📝 **핵심 개념 정리**

### **클래스와 객체**

- 클래스: 설계도 (붕어빵 틀)
- 객체: 실제 제품 (붕어빵)

### **클래스 만들기**

```python
class ClassName:
    def __init__(self, params):
        self.attribute = value
  
    def method(self):
        # 기능 구현
        pass
```

### **핵심 용어**

- `__init__`: 생성자, 객체 초기화
- `self`: 객체 자신을 가리킴
- 속성(attribute): 데이터
- 메서드(method): 기능

### **변수 종류**

- 인스턴스 변수: `self.name` (객체마다 다름)
- 클래스 변수: `ClassName.count` (모든 객체가 공유)
