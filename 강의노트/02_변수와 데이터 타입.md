## 1️⃣ **변수란 무엇인가?**

변수(Variable)는 프로그래밍에서 데이터를 저장하는 공간입니다. 마치 우리가 물건을 보관하기 위해 상자에 라벨을 붙이듯이, 변수는 데이터를 담고 있는 상자에 이름을 붙인 것이라고 생각하면 됩니다. 예를 들어, 학생의 점수를 저장하려면 `score`라는 이름의 변수를 만들고 그 안에 점수 값을 넣을 수 있습니다.

변수를 사용하면 같은 값을 여러 번 입력할 필요 없이 변수 이름만으로 그 값을 계속 사용할 수 있습니다. 또한 프로그램이 실행되는 동안 변수에 저장된 값을 바꿀 수도 있습니다. 이것이 바로 "변수"라고 부르는 이유입니다 - 값이 변할 수 있기 때문입니다.

```
변수의 개념

┌─────────────────────────┐
│   변수 이름: age         │  ← 라벨 (우리가 부르는 이름)
│  ┌───────────────────┐  │
│  │                   │  │
│  │        20         │  │  ← 저장된 값
│  │                   │  │
│  └───────────────────┘  │
│    메모리 공간            │
└─────────────────────────┘
```

### **변수 선언과 할당**

파이썬에서 변수를 만드는 것은 매우 간단합니다. 등호(`=`)를 사용하여 변수에 값을 할당하기만 하면 됩니다. 이 등호는 수학에서의 같다는 의미가 아니라 "오른쪽 값을 왼쪽 변수에 저장하라"는 의미입니다.

```python
# 변수 선언과 값 할당
age = 20
name = "김철수"
height = 175.5
is_student = True
```

위 코드에서 `age`라는 변수에 숫자 20을 저장했고, `name`이라는 변수에 문자열 "김철수"를 저장했습니다. 한 번에 여러 변수를 선언할 수도 있습니다.

```python
# 여러 변수를 한 줄에 선언
x, y, z = 10, 20, 30

# 같은 값을 여러 변수에 할당
a = b = c = 100
```

변수에 저장된 값은 언제든지 바꿀 수 있습니다. 이것이 변수의 핵심 기능입니다.

```python
# 변수 값 변경하기
score = 85
print("처음 점수:", score)

score = 95  # 점수를 95로 변경
print("변경된 점수:", score)
```

## 2️⃣ **변수 이름 짓기 규칙**

변수 이름을 지을 때는 반드시 따라야 하는 규칙들이 있습니다. 이 규칙을 어기면 프로그램이 실행되지 않습니다.

첫째, 변수 이름은 영문자(a-z, A-Z), 숫자(0-9), 밑줄(_)만 사용할 수 있습니다. 한글도 기술적으로는 가능하지만 권장하지 않습니다. 특수문자(@, #, $ 등)는 사용할 수 없습니다.

둘째, 변수 이름은 숫자로 시작할 수 없습니다. `1st_place`는 잘못된 이름이지만, `first_place`나 `place_1`은 올바른 이름입니다.

셋째, 파이썬의 예약어(keyword)는 변수 이름으로 사용할 수 없습니다. 예약어란 파이썬에서 이미 특별한 의미로 사용하고 있는 단어들입니다. 예를 들어 `if`, `for`, `while`, `print`, `True`, `False` 같은 단어들입니다.

넷째, 변수 이름은 대소문자를 구분합니다. `name`, `Name`, `NAME`은 모두 다른 변수입니다.

```python
# 올바른 변수 이름
student_name = "홍길동"
total_score = 100
_private_var = 42
myAge = 25

# 잘못된 변수 이름 (오류 발생)
# 2nd_student = "김영희"  # 숫자로 시작
# my-name = "이철수"      # 하이픈 사용 불가
# for = 10               # 예약어 사용 불가
```

### **좋은 변수 이름 짓기**

규칙을 따르는 것만으로는 충분하지 않습니다. 다른 사람이 코드를 읽었을 때, 또는 나중에 자신이 다시 코드를 볼 때 쉽게 이해할 수 있도록 의미 있는 이름을 사용해야 합니다.

```python
# 나쁜 예 - 의미를 알 수 없음
a = 85
b = 90
c = 88

# 좋은 예 - 의미가 명확함
korean_score = 85
english_score = 90
math_score = 88
```

파이썬에서는 일반적으로 소문자와 밑줄을 사용하는 스네이크 케이스(snake_case)를 권장합니다. 여러 단어를 연결할 때 밑줄로 구분하는 방식입니다.

```python
# 스네이크 케이스 (Python 권장)
student_name = "김영희"
total_count = 100
max_temperature = 35.5

# 카멜 케이스 (다른 언어에서 주로 사용)
studentName = "김영희"
totalCount = 100
maxTemperature = 35.5
```

## 3️⃣ **데이터 타입의 종류**

파이썬에서 변수에 저장할 수 있는 데이터는 여러 종류(타입)가 있습니다. 각 타입마다 저장할 수 있는 값의 형태와 사용 목적이 다릅니다. 마치 우리가 음료는 컵에, 쌀은 쌀통에 담듯이 데이터도 그 성격에 맞는 타입으로 저장해야 합니다.

```
주요 데이터 타입

┌──────────────────────────────────────┐
│  정수 (int)        │  10, -5, 1000    │
├──────────────────────────────────────┤
│  실수 (float)      │  3.14, -0.5, 2.0 │
├──────────────────────────────────────┤
│  문자열 (str)      │  "Hello", '안녕'  │
├──────────────────────────────────────┤
│  불린 (bool)       │  True, False     │
└──────────────────────────────────────┘
```

### **정수형 (Integer, int)**

정수형은 소수점이 없는 숫자를 저장하는 타입입니다. 나이, 개수, 순위 등을 표현할 때 사용합니다. 양수, 음수, 0 모두 정수형에 포함됩니다.

```python
# 정수형 변수 예제
age = 20              # 나이
student_count = 35    # 학생 수
temperature = -5      # 영하 5도
year = 2024          # 연도
```

정수는 아무리 큰 수도 저장할 수 있습니다. 다른 프로그래밍 언어와 달리 파이썬은 정수의 크기에 제한이 없습니다.

```python
# 큰 숫자도 문제없이 저장
population = 51000000           # 인구 5천1백만
national_debt = 1000000000000   # 1조
```

### **실수형 (Floating Point, float)**

실수형은 소수점이 있는 숫자를 저장하는 타입입니다. 키, 몸무게, 온도, 금액 등 정확한 값이 필요할 때 사용합니다.

```python
# 실수형 변수 예제
height = 175.5        # 키 175.5cm
weight = 68.3         # 몸무게 68.3kg
pi = 3.14159         # 원주율
temperature = 36.5    # 체온
```

실수는 과학적 표기법으로도 표현할 수 있습니다. 매우 크거나 작은 숫자를 표현할 때 유용합니다.

```python
# 과학적 표기법
speed_of_light = 3.0e8    # 3.0 × 10^8 = 300,000,000
small_number = 1.5e-3     # 1.5 × 10^-3 = 0.0015
```

### **문자열 (String, str)**

문자열은 텍스트 데이터를 저장하는 타입입니다. 이름, 주소, 메시지 등 문자로 이루어진 모든 것을 저장할 수 있습니다. 문자열은 큰따옴표(`"`) 또는 작은따옴표(`'`)로 감싸서 표현합니다.

```python
# 문자열 변수 예제
name = "김철수"
address = '서울시 강남구'
message = "안녕하세요!"
empty_string = ""     # 빈 문자열도 가능
```

큰따옴표와 작은따옴표는 기능상 차이가 없지만, 문자열 안에 따옴표를 포함할 때 유용하게 사용할 수 있습니다.

```python
# 문자열 안에 따옴표 포함하기
sentence1 = "He said, 'Hello!'"
sentence2 = 'She said, "Goodbye!"'
```

여러 줄의 문자열을 저장하려면 삼중 따옴표(`"""` 또는 `'''`)를 사용합니다.

```python
# 여러 줄 문자열
poem = """
동해물과 백두산이
마르고 닳도록
하느님이 보우하사
우리나라 만세
"""
```

### **불린형 (Boolean, bool)**

불린형은 참(True) 또는 거짓(False) 두 가지 값만 가질 수 있는 타입입니다. 조건을 판단하거나 상태를 표현할 때 사용합니다. 불린값은 반드시 첫 글자를 대문자로 써야 합니다.

```python
# 불린형 변수 예제
is_student = True       # 학생인가?
is_adult = False        # 성인인가?
has_license = True      # 면허가 있는가?
is_raining = False      # 비가 오는가?
```

불린형은 주로 비교 연산이나 조건문에서 많이 사용됩니다.

```python
# 비교 연산의 결과는 불린형
age = 20
is_adult = age >= 18    # True (20은 18 이상이므로)
is_teenager = age < 13  # False (20은 13보다 작지 않으므로)
```

## 4️⃣ **타입 확인하기**

변수에 저장된 데이터가 어떤 타입인지 확인하려면 `type()` 함수를 사용합니다. 이 함수는 프로그램을 작성할 때 매우 유용합니다.

```python
# 다양한 타입 확인하기
age = 20
height = 175.5
name = "김영희"
is_student = True

print(type(age))         # <class 'int'>
print(type(height))      # <class 'float'>
print(type(name))        # <class 'str'>
print(type(is_student))  # <class 'bool'>
```

### **예제 1: 변수 타입 확인 프로그램**

실생활에서 회원 가입 정보를 입력받는 상황을 생각해봅시다. 각 정보의 타입을 확인하는 프로그램을 만들어봅시다.

```python
# 회원 정보 입력 프로그램
print("=== 회원 정보 입력 ===")

# 회원 정보 저장
member_id = 1001
member_name = "홍길동"
age = 25
height = 178.5
weight = 72.3
is_premium = True

# 정보 출력
print(f"회원 번호: {member_id} (타입: {type(member_id).__name__})")
print(f"이름: {member_name} (타입: {type(member_name).__name__})")
print(f"나이: {age} (타입: {type(age).__name__})")
print(f"키: {height}cm (타입: {type(height).__name__})")
print(f"몸무게: {weight}kg (타입: {type(weight).__name__})")
print(f"프리미엄 회원: {is_premium} (타입: {type(is_premium).__name__})")
```

## 5️⃣ **타입 변환하기**

때로는 한 타입의 데이터를 다른 타입으로 변환해야 할 때가 있습니다. 예를 들어, 사용자로부터 입력받은 문자열을 숫자로 바꿔서 계산해야 하는 경우입니다. 이를 타입 변환(Type Conversion) 또는 타입 캐스팅(Type Casting)이라고 합니다.

### **정수로 변환: int()**

문자열이나 실수를 정수로 변환할 때 `int()` 함수를 사용합니다. 실수를 정수로 변환하면 소수점 이하는 버려집니다(반올림하지 않음).

```python
# 문자열을 정수로 변환
age_str = "25"
age_int = int(age_str)
print(age_int)        # 25
print(type(age_int))  # <class 'int'>

# 실수를 정수로 변환 (소수점 버림)
height = 175.8
height_int = int(height)
print(height_int)     # 175 (175.8이 아님)
```

### **실수로 변환: float()**

문자열이나 정수를 실수로 변환할 때 `float()` 함수를 사용합니다.

```python
# 문자열을 실수로 변환
price_str = "1500.50"
price_float = float(price_str)
print(price_float)    # 1500.5

# 정수를 실수로 변환
count = 10
count_float = float(count)
print(count_float)    # 10.0
```

### **문자열로 변환: str()**

숫자를 문자열로 변환할 때 `str()` 함수를 사용합니다. 출력이나 문자열 연결에 유용합니다.

```python
# 정수를 문자열로 변환
age = 20
age_str = str(age)
print(age_str)        # "20"
print(type(age_str))  # <class 'str'>

# 문자열 연결에 활용
message = "제 나이는 " + str(age) + "살입니다."
print(message)        # "제 나이는 20살입니다."
```

### **예제 2: 쇼핑몰 가격 계산 프로그램**

실생활에서 온라인 쇼핑을 할 때 상품 가격과 수량을 입력받아 총 금액을 계산하는 프로그램을 만들어봅시다.

```python
# 쇼핑몰 가격 계산 프로그램
print("=== 온라인 쇼핑 계산기 ===")

# 사용자 입력 (input()은 항상 문자열 반환)
product_name = input("상품명을 입력하세요: ")
price_input = input("상품 가격을 입력하세요: ")
quantity_input = input("구매 수량을 입력하세요: ")

# 문자열을 숫자로 변환
price = int(price_input)
quantity = int(quantity_input)

# 총 금액 계산
total_price = price * quantity

# 결과 출력
print("\n=== 구매 내역 ===")
print(f"상품명: {product_name}")
print(f"단가: {price:,}원")
print(f"수량: {quantity}개")
print(f"총 금액: {total_price:,}원")
```

### **예제 3: 성적 평균 계산기**

학생의 세 과목 점수를 입력받아 평균을 계산하는 프로그램입니다. 평균은 소수점까지 표시해야 하므로 실수로 계산합니다.

```python
# 성적 평균 계산 프로그램
print("=== 성적 평균 계산기 ===")

# 학생 정보 입력
student_name = input("학생 이름: ")

# 점수 입력 (문자열을 정수로 변환)
korean = int(input("국어 점수: "))
english = int(input("영어 점수: "))
math = int(input("수학 점수: "))

# 평균 계산 (정수를 실수로 변환하여 정확한 평균 계산)
total = korean + english + math
average = total / 3  # 자동으로 실수 결과 반환

# 결과 출력
print(f"\n{student_name} 학생의 성적")
print(f"국어: {korean}점")
print(f"영어: {english}점")
print(f"수학: {math}점")
print(f"총점: {total}점")
print(f"평균: {average:.2f}점")  # 소수점 둘째 자리까지 표시
```

## 6️⃣ **변수 사용시 주의사항**

변수를 사용할 때 몇 가지 주의해야 할 점이 있습니다. 이러한 점들을 알아두면 오류를 예방하고 더 안전한 프로그램을 작성할 수 있습니다.

### **변수 초기화**

변수를 사용하기 전에 반드시 값을 할당해야 합니다. 선언하지 않은 변수를 사용하면 오류가 발생합니다.

```python
# 잘못된 예 - 초기화하지 않은 변수 사용
# print(score)  # NameError 발생

# 올바른 예 - 변수 초기화 후 사용
score = 0
print(score)  # 0
```

### **타입 변환 주의사항**

모든 문자열이 숫자로 변환될 수 있는 것은 아닙니다. 숫자가 아닌 문자열을 `int()`나 `float()`로 변환하려고 하면 오류가 발생합니다.

```python
# 올바른 변환
num1 = int("100")      # 정상 작동
num2 = float("3.14")   # 정상 작동

# 잘못된 변환 (오류 발생)
# num3 = int("안녕")    # ValueError 발생
# num4 = float("abc")   # ValueError 발생
```

### **정수 나눗셈**

파이썬에서 나눗셈(`/`)은 항상 실수 결과를 반환합니다. 정수 나눗셈이 필요하면 `//` 연산자를 사용합니다.

```python
# 일반 나눗셈 (실수 결과)
result1 = 10 / 3
print(result1)  # 3.3333333333333335

# 정수 나눗셈 (몫만 반환)
result2 = 10 // 3
print(result2)  # 3

# 나머지 연산
result3 = 10 % 3
print(result3)  # 1
```

## 📝 **핵심 개념 정리**

변수는 데이터를 저장하는 공간으로, 등호를 사용하여 값을 할당합니다. 변수 이름은 영문자, 숫자, 밑줄만 사용할 수 있으며 숫자로 시작할 수 없고, 예약어는 사용할 수 없습니다. 의미 있는 변수 이름을 사용하면 코드의 가독성이 높아집니다.

파이썬의 주요 데이터 타입에는 정수형(int), 실수형(float), 문자열(str), 불린형(bool)이 있습니다. 각 타입은 서로 다른 종류의 데이터를 저장하며, `type()` 함수로 변수의 타입을 확인할 수 있습니다.

타입 변환은 `int()`, `float()`, `str()` 함수를 사용하여 수행합니다. 특히 `input()` 함수는 항상 문자열을 반환하므로, 숫자 계산을 하려면 적절한 타입으로 변환해야 합니다. 변수는 사용하기 전에 반드시 초기화해야 하며, 숫자가 아닌 문자열을 숫자로 변환하려고 하면 오류가 발생합니다.

## 🔑 **퀴즈 정답 및 해설**

**1번 정답: 3**
`student_age`가 올바른 변수 이름입니다. (1) `1st_place`는 숫자로 시작하므로 불가능, (2) `my-name`은 하이픈을 사용할 수 없음, (4) `for`는 파이썬 예약어이므로 사용 불가능합니다.

**2번 정답: 2**
