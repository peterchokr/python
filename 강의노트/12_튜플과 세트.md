## 1️⃣ **튜플(Tuple)이란?**

튜플은 리스트와 비슷하지만 **한 번 만들면 변경할 수 없습니다** (불변, immutable).

### **튜플 vs 리스트**

```python
# 리스트 (변경 가능)
my_list = [1, 2, 3]
my_list[0] = 10  # 가능
print(my_list)  # [10, 2, 3]

# 튜플 (변경 불가능)
my_tuple = (1, 2, 3)
# my_tuple[0] = 10  # 오류 발생!
```

```
리스트 vs 튜플

┌─────────────┬─────────────┬─────────────┐
│   특징      │   리스트    │   튜플      │
├─────────────┼─────────────┼─────────────┤
│  괄호       │    [ ]      │    ( )      │
│  변경 가능  │     O       │     X       │
│  속도       │   보통      │   빠름      │
│  사용 시기  │ 변경 필요   │ 변경 불필요 │
└─────────────┴─────────────┴─────────────┘
```

### **튜플 생성하기**

```python
# 소괄호로 생성
coordinates = (37.5, 127.0)  # 좌표
rgb = (255, 128, 0)          # RGB 색상

# 괄호 없이도 가능
point = 10, 20

# 항목이 하나인 튜플 (쉼표 필수!)
single = (42,)  # 튜플
not_tuple = (42)  # 그냥 숫자

print(type(single))     # <class 'tuple'>
print(type(not_tuple))  # <class 'int'>
```

### **예제 1: 도시 정보**

도시의 고정 정보를 튜플로 저장하는 프로그램입니다.

```python
# 도시 정보 프로그램
print("🌆" + "=" * 38 + "🌆")
print("   세계 도시 정보")
print("🌆" + "=" * 38 + "🌆")

# 도시 정보 (이름, 국가, 인구, 면적)
seoul = ("서울", "대한민국", 9776000, 605)
tokyo = ("도쿄", "일본", 13960000, 2194)
newyork = ("뉴욕", "미국", 8336000, 783)

cities = [seoul, tokyo, newyork]

print("\n도시 정보:")
print("=" * 40)

for city in cities:
    name, country, population, area = city  # 튜플 언패킹
    density = population / area
  
    print(f"\n📍 {name} ({country})")
    print(f"   인구: {population:,}명")
    print(f"   면적: {area}km²")
    print(f"   인구밀도: {density:,.0f}명/km²")

print("=" * 40)
```

## 2️⃣ **튜플 언패킹**

튜플의 값을 여러 변수에 한 번에 할당할 수 있습니다.

```python
# 튜플 언패킹
point = (10, 20)
x, y = point
print(x, y)  # 10 20

# 여러 값 동시 할당
name, age, city = ("김철수", 25, "서울")
print(f"{name}, {age}세, {city}")
```

### **값 교환하기**

```python
# 튜플 언패킹으로 값 교환
a = 10
b = 20

# 간단한 교환
a, b = b, a
print(a, b)  # 20 10
```

### **예제 2: 여행 일정**

여행 일정을 튜플로 관리하는 프로그램입니다.

```python
# 여행 일정 관리 프로그램
print("✈️" + "=" * 38 + "✈️")
print("   여행 일정표")
print("✈️" + "=" * 38 + "✈️")

# 일정 (날짜, 도시, 활동, 예산)
```

## 3️⃣ **세트(Set)란?**

세트는 **중복을 허용하지 않는** 자료구조입니다. 수학의 집합과 같은 개념입니다.

### **세트 생성하기**

```python
# 중괄호로 생성
fruits = {"사과", "바나나", "오렌지"}
print(fruits)  # {'사과', '바나나', '오렌지'}

# set() 함수로 생성
numbers = set([1, 2, 3, 2, 1])
print(numbers)  # {1, 2, 3} - 중복 자동 제거!

# 빈 세트
empty_set = set()  # {} 는 빈 딕셔너리!
```

⚠️ **주의**: 세트는 **순서가 없습니다**. 인덱스로 접근할 수 없습니다!

### **예제 3: 중복 제거**

설문조사 응답에서 중복을 제거하는 프로그램입니다.

```python
# 설문조사 중복 제거 프로그램
print("📊" + "=" * 38 + "📊")
print("   설문조사 분석")
print("📊" + "=" * 38 + "📊")

responses = []

print("\n좋아하는 음료를 입력하세요 (종료: q)")

while True:
    drink = input("음료: ")
  
    if drink == "q" or drink == "Q":
        break
  
    responses.append(drink)
    print(f"✓ '{drink}' 기록됨")

# 중복 제거
unique_drinks = set(responses)

# 결과
print("\n" + "=" * 40)
print("설문 결과")
print("=" * 40)
print(f"총 응답 수: {len(responses)}개")
print(f"고유 음료 수: {len(unique_drinks)}개")

print("\n음료별 투표 수:")
print("-" * 40)
for drink in unique_drinks:
    count = responses.count(drink)
    print(f"• {drink}: {count}표")

print("=" * 40)
```

## 4️⃣ **세트에 항목 추가/제거**

### **add() - 항목 추가**

```python
# add()
fruits = {"사과", "바나나"}
fruits.add("오렌지")
print(fruits)  # {'사과', '바나나', '오렌지'}

# 이미 있는 항목 추가 (무시됨)
fruits.add("사과")
print(fruits)  # {'사과', '바나나', '오렌지'} - 변화 없음
```

### **remove() / discard() - 항목 제거**

```python
# remove() - 항목 제거 (없으면 오류)
colors = {"빨강", "파랑", "초록"}
colors.remove("파랑")
print(colors)  # {'빨강', '초록'}

# discard() - 항목 제거 (없어도 오류 안 남)
colors.discard("노랑")  # 오류 안 남
print(colors)  # {'빨강', '초록'}
```

### **예제 4: 참석자 관리**

행사 참석자를 관리하는 프로그램입니다.

```python
# 행사 참석자 관리 프로그램
print("🎉" + "=" * 38 + "🎉")
print("   행사 참석자 관리")
print("🎉" + "=" * 38 + "🎉")

attendees = set()

while True:
    print("\n" + "=" * 40)
    print("1. 참석 등록")
    print("2. 등록 취소")
    print("3. 참석자 확인")
    print("4. 전체 명단")
    print("5. 종료")
    print("=" * 40)
  
    choice = input("\n선택: ")
  
    if choice == "1":
        name = input("\n이름: ")
        if name in attendees:
            print(f"'{name}'님은 이미 등록되어 있습니다.")
        else:
            attendees.add(name)
            print(f"✓ '{name}' 등록 완료")
  
    elif choice == "2":
        name = input("\n취소할 이름: ")
        if name in attendees:
            attendees.discard(name)
            print(f"✓ '{name}' 취소됨")
        else:
            print(f"'{name}'님은 등록되지 않았습니다.")
  
    elif choice == "3":
        name = input("\n확인할 이름: ")
        if name in attendees:
            print(f"✓ '{name}'님은 참석 예정입니다.")
        else:
            print(f"'{name}'님은 명단에 없습니다.")
  
    elif choice == "4":
        print("\n👥 참석자 명단")
        print("-" * 40)
        if len(attendees) == 0:
            print("등록된 참석자가 없습니다.")
        else:
            for i, name in enumerate(attendees, 1):
                print(f"{i}. {name}")
        print("-" * 40)
        print(f"총 {len(attendees)}명")
  
    elif choice == "5":
        print("\n프로그램을 종료합니다.")
        break
  
    else:
        print("잘못된 선택입니다.")
```

## 5️⃣ **세트 연산**

세트는 수학의 집합 연산을 지원합니다.

### **합집합 (Union)**

```python
# | 연산자
a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)  # {1, 2, 3, 4, 5}

# union() 메서드
print(a.union(b))  # {1, 2, 3, 4, 5}
```

### **교집합 (Intersection)**

```python
# & 연산자
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}
print(a & b)  # {3, 4}

# intersection() 메서드
print(a.intersection(b))  # {3, 4}
```

### **차집합 (Difference)**

```python
# - 연산자
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}
print(a - b)  # {1, 2} - a에만 있는 것
print(b - a)  # {5, 6} - b에만 있는 것

# difference() 메서드
print(a.difference(b))  # {1, 2}
```

### **예제 5: 영화 취향 비교**

두 사람의 영화 취향을 분석하는 프로그램입니다.

```python
# 영화 취향 비교 프로그램
print("🎬" + "=" * 38 + "🎬")
print("   영화 취향 분석")
print("🎬" + "=" * 38 + "🎬")

# 첫 번째 사람
print("\n첫 번째 사람 이름:")
name1 = input("이름: ")

print(f"\n{name1}님이 좋아하는 영화 장르 (종료: q):")
genres1 = set()

while True:
    genre = input("장르: ")
    if genre == "q" or genre == "Q":
        break
    genres1.add(genre)

# 두 번째 사람
print("\n두 번째 사람 이름:")
name2 = input("이름: ")

print(f"\n{name2}님이 좋아하는 영화 장르 (종료: q):")
genres2 = set()

while True:
    genre = input("장르: ")
    if genre == "q" or genre == "Q":
        break
    genres2.add(genre)

# 분석
common = genres1 & genres2  # 공통 장르
only_person1 = genres1 - genres2  # 첫 번째만
only_person2 = genres2 - genres1  # 두 번째만
all_genres = genres1 | genres2  # 전체 장르

# 결과
print("\n" + "=" * 40)
print("분석 결과")
print("=" * 40)

print(f"\n{name1}님의 장르:")
for genre in genres1:
    print(f"  • {genre}")

print(f"\n{name2}님의 장르:")
for genre in genres2:
    print(f"  • {genre}")

print("\n❤️ 공통 관심 장르:")
if len(common) == 0:
    print("  공통 장르가 없습니다.")
else:
    for genre in common:
        print(f"  • {genre}")

print(f"\n💡 {name1}님만 좋아하는 장르:")
if len(only_person1) == 0:
    print("  없음")
else:
    for genre in only_person1:
        print(f"  • {genre}")

print(f"\n💡 {name2}님만 좋아하는 장르:")
if len(only_person2) == 0:
    print("  없음")
else:
    for genre in only_person2:
        print(f"  • {genre}")

print(f"\n📊 통합 장르 (총 {len(all_genres)}개):")
for genre in all_genres:
    print(f"  • {genre}")

print("=" * 40)
```

## 6️⃣ **부분집합과 상위집합**

```python
# 부분집합
a = {1, 2}
b = {1, 2, 3, 4}

# issubset() - a가 b의 부분집합인가?
print(a.issubset(b))  # True

# issuperset() - b가 a의 상위집합인가?
print(b.issuperset(a))  # True

# <= 연산자
print(a <= b)  # True (부분집합)
print(b >= a)  # True (상위집합)
```

## 📝 **핵심 개념 정리**

**튜플**은 소괄호 `()`로 만들며, 한 번 만들면 변경할 수 없는 불변 자료구조입니다. 튜플 언패킹으로 여러 변수에 값을 동시에 할당할 수 있습니다. 변경하지 않을 고정 데이터를 저장할 때 유용합니다.

**세트**는 중괄호 `{}`나 `set()` 함수로 만들며, 중복을 허용하지 않고 순서가 없는 자료구조입니다. `add()`로 항목을 추가하고, `remove()`나 `discard()`로 제거합니다.

세트는 합집합(`|`), 교집합(`&`), 차집합(`-`) 등의 집합 연산을 지원합니다. `issubset()`과 `issuperset()`으로 부분집합과 상위집합을 확인할 수 있습니다.

## 🔑 **퀴즈 정답 및 해설**

**1번 정답: 2**
튜플은 한 번 만들면 변경할 수 없는 불변(immutable) 자료구조입니다.

**2번 정답: 1**

**3번 정답: 2**
`&`는 교집합 연산자로, 두 세트의 공통 원소인 {3}을 반환합니다.

**5번 정답: 1**
