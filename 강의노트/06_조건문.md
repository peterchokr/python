## 1️⃣ **if 문 기본**

조건문은 특정 조건이 참일 때만 코드를 실행하는 구조입니다. 마치 "만약 비가 오면 우산을 가져간다"처럼 조건에 따라 행동을 결정하는 것과 같습니다.

### **기본 구조**

```python
# if 문의 기본 형태
if 조건식:
    실행할 코드
```

조건식이 참(True)이면 들여쓰기된 코드가 실행되고, 거짓(False)이면 실행되지 않습니다. 파이썬에서 들여쓰기는 매우 중요하며, 일반적으로 스페이스 4칸을 사용합니다.

```python
# 나이 확인
age = int(input("나이를 입력하세요: "))

if age >= 18:
    print("성인입니다.")
    print("모든 영화를 관람할 수 있습니다.")

print("프로그램을 종료합니다.")
```

위 예제에서 나이가 18세 이상이면 "성인입니다."와 "모든 영화를 관람할 수 있습니다."가 출력됩니다. 마지막 줄은 조건과 관계없이 항상 실행됩니다.

```
조건문 실행 흐름

       시작
        │
        ▼
   [조건 검사]
    age >= 18?
      /    \
   True   False
    /        \
   ▼          │
실행 블록      │
(들여쓰기)     │
   │          │
   ▼          ▼
   다음 코드 실행
```

### **예제 1: 카페 모닝 할인**

시간에 따라 할인을 적용하는 카페 주문 프로그램입니다.

```python
# 카페 모닝 할인 시스템
print("☕ === 카페 주문 시스템 === ☕\n")

# 주문 정보
drink = input("음료: ")
price = int(input("가격: "))
hour = int(input("현재 시간 (0-23): "))

print("\n" + "=" * 40)
print("주문 내역")
print("=" * 40)
print(f"음료: {drink}")
print(f"정가: {price:,}원")

# 모닝 할인 (오전 6시 ~ 10시)
if hour >= 6 and hour < 10:
    discount = int(price * 0.3)  # 30% 할인
    price = price - discount
    print(f"모닝 할인 30%: -{discount:,}원")

print(f"최종 가격: {price:,}원")
print("=" * 40)

if hour >= 6 and hour < 10:
    print("🌅 좋은 아침입니다!")
```

## 2️⃣ **if-else 문**

조건이 참일 때와 거짓일 때 각각 다른 동작을 하려면 `else`를 사용합니다.

```python
# if-else 기본 구조
if 조건식:
    조건이 참일 때 실행
else:
    조건이 거짓일 때 실행
```

```python
# 영화 관람 등급 확인
age = int(input("나이: "))

if age >= 18:
    print("성인 인증 완료")
    print("19세 관람가 영화를 볼 수 있습니다.")
else:
    print("미성년자입니다.")
    print("청소년 관람가 영화만 볼 수 있습니다.")
```

### **예제 2: 배달비 계산**

주문 금액에 따라 배달비를 다르게 적용하는 프로그램입니다.

```python
# 배달비 계산 시스템
print("🚚 === 배달 주문 시스템 === 🚚\n")

# 주문 정보
restaurant = input("식당 이름: ")
```

## 3️⃣ **if-elif-else 문**

여러 조건을 순차적으로 검사할 때는 `elif`(else if의 줄임말)를 사용합니다.

```python
# if-elif-else 기본 구조
if 조건1:
    조건1이 참일 때
elif 조건2:
    조건1은 거짓, 조건2가 참일 때
elif 조건3:
    조건1,2는 거짓, 조건3이 참일 때
else:
    모든 조건이 거짓일 때
```

조건은 위에서부터 순서대로 검사되며, 처음으로 참인 조건의 블록만 실행됩니다. 한 번 조건이 만족되면 나머지 조건은 검사하지 않습니다.

```python
# 성적 등급 판정
score = int(input("점수: "))

if score >= 90:
    grade = "A"
    print("A학점입니다. 우수합니다!")
elif score >= 80:
    grade = "B"
    print("B학점입니다. 잘했습니다!")
elif score >= 70:
    grade = "C"
    print("C학점입니다. 노력하세요!")
elif score >= 60:
    grade = "D"
    print("D학점입니다. 분발하세요!")
else:
    grade = "F"
    print("F학점입니다. 재수강하세요!")

print(f"최종 등급: {grade}")
```

## 4️⃣ **중첩 조건문**

조건문 안에 또 다른 조건문을 넣을 수 있습니다. 이를 중첩 조건문이라고 합니다.

```python
# 중첩 조건문 기본 구조
if 조건1:
    if 조건2:
        조건1과 조건2가 모두 참일 때
    else:
        조건1은 참, 조건2는 거짓일 때
else:
    조건1이 거짓일 때
```

```python
# 영화 티켓 가격 계산
age = int(input("나이: "))
is_student = input("학생인가요? (yes/no): ")

price = 14000  # 기본 가격

if age < 18:
    price = 11000  # 청소년 가격
    print("청소년 요금이 적용됩니다.")
else:
    if is_student == "yes":
        price = 12000  # 성인 학생 할인
        print("성인 학생 할인이 적용됩니다.")
    else:
        print("일반 요금이 적용됩니다.")

print(f"티켓 가격: {price:,}원")
```

## 5️⃣ **조건 표현식 (삼항 연산자)**

간단한 if-else 문은 한 줄로 표현할 수 있습니다. 이를 조건 표현식 또는 삼항 연산자라고 합니다.

```python
# 조건 표현식 기본 구조
변수 = 참일_때_값 if 조건 else 거짓일_때_값
```

```python
# 일반적인 if-else
age = 20
if age >= 18:
    status = "성인"
else:
    status = "미성년자"

# 조건 표현식으로 간단하게
age = 20
status = "성인" if age >= 18 else "미성년자"
print(status)  # 성인

# 최댓값 구하기
a = 10
b = 20
max_value = a if a > b else b
print(f"최댓값: {max_value}")  # 20

# 절댓값 구하기
num = -5
abs_num = num if num >= 0 else -num
print(f"절댓값: {abs_num}")  # 5
```

### **예제 4: 유튜버 등급 판정**

구독자 수에 따라 유튜버 등급을 판정하는 프로그램입니다.

```python
# 유튜버 등급 판정 시스템
print("📺 === 유튜버 통계 시스템 === 📺\n")

# 채널 정보
channel_name = input("채널 이름: ")
subscribers = int(input("구독자 수: "))
views = int(input("총 조회수: "))
videos = int(input("영상 개수: "))

# 평균 조회수
avg_views = views / videos if videos > 0 else 0

# 등급 판정
if subscribers >= 1000000:
    tier = "💎 다이아몬드"
    message = "축하합니다! 최상위 유튜버입니다!"
elif subscribers >= 100000:
    tier = "🏆 골드"
    message = "훌륭해요! 인기 유튜버입니다!"
elif subscribers >= 10000:
    tier = "🥈 실버"
    message = "잘하고 있어요! 성장 중입니다!"
elif subscribers >= 1000:
    tier = "🥉 브론즈"
    message = "좋은 시작이에요!"
else:
    tier = "🌱 새싹"
    message = "화이팅! 꾸준히 업로드하세요!"

# 결과 출력
print("\n" + "╔" + "═" * 48 + "╗")
print("║" + f"{channel_name} 채널 통계".center(48) + "║")
print("╠" + "═" + "═" * 48 + "╣")
print(f"║  구독자: {subscribers:,}명".ljust(50) + "║")
print(f"║  총 조회수: {views:,}회".ljust(50) + "║")

print(f"║  평균 조회수: {avg_views:,.0f}회".ljust(50) + "║")
print("╠" + "─" * 50 + "╣")
print(f"║  등급: {tier}".ljust(50) + "║")
print(f"║  {message}".ljust(50) + "║")

# 다음 등급까지
if subscribers < 1000:
    next_goal = 1000 - subscribers
    print(f"║  브론즈까지 {next_goal:,}명 남았어요!".ljust(50) + "║")
elif subscribers < 10000:
    next_goal = 10000 - subscribers
    print(f"║  실버까지 {next_goal:,}명 남았어요!".ljust(50) + "║")
elif subscribers < 100000:
    next_goal = 100000 - subscribers
    print(f"║  골드까지 {next_goal:,}명 남았어요!".ljust(50) + "║")
elif subscribers < 1000000:
    next_goal = 1000000 - subscribers
    print(f"║  다이아몬드까지 {next_goal:,}명 남았어요!".ljust(50) + "║")

print("╚" + "═" * 50 + "╝")
```

## 6️⃣ **논리 연산자와 조건문**

복잡한 조건을 만들 때는 논리 연산자 `and`, `or`, `not`을 활용합니다.

```python
# and: 모두 참이어야 참
age = 25
has_license = True

if age >= 18 and has_license:
    print("운전 가능합니다.")

# or: 하나라도 참이면 참
is_weekend = True
is_holiday = False

if is_weekend or is_holiday:
    print("쉬는 날입니다!")

# not: 참과 거짓을 반대로
is_raining = False

if not is_raining:
    print("우산이 필요 없습니다.")
```

### **예제 5: 영화 예매 시스템**

나이, 시간, 좌석을 고려한 영화 예매 시스템입니다.

```python
# 영화 예매 시스템
print("🎬 === 영화 예매 시스템 === 🎬\n")

# 영화 정보
movie = input("영화 제목: ")
rating = input("관람등급 (전체/12세/15세/청불): ")

# 고객 정보
name = input("\n예매자 이름: ")
age = int(input("나이: "))
is_student = input("학생인가요? (yes/no): ")

# 예매 정보
time = int(input("\n상영 시간 (0-23시): "))
seat_type = input("좌석 등급 (일반/VIP): ")
tickets = int(input("티켓 수: "))

# 기본 가격
base_price = 14000

# 좌석 등급별 가격
if seat_type == "VIP":
    base_price = 18000

# 나이별 할인
if age < 13:
    base_price = 10000  # 어린이
elif age >= 65:
    base_price = 11000  # 경로우대
elif is_student == "yes":
    base_price = base_price - 2000  # 학생 할인

# 조조 할인 (6시-10시)
morning_discount = 0
if time >= 6 and time < 10:
    morning_discount = 2000
    base_price = base_price - morning_discount

# 총 가격
total_price = base_price * tickets

# 관람 가능 여부 확인
can_watch = True
reason = ""

if rating == "청불" and age < 18:
    can_watch = False
    reason = "청소년 관람불가 영화입니다."
elif rating == "15세" and age < 15:
    can_watch = False
    reason = "15세 미만은 관람할 수 없습니다."
elif rating == "12세" and age < 12:
    can_watch = False
    reason = "12세 미만은 관람할 수 없습니다."

# 결과 출력
print("\n" + "=" * 50)
if can_watch:
    print("✅ 예매 완료!")
    print("=" * 50)
    print(f"영화: {movie} ({rating})")
    print(f"예매자: {name} ({age}세)")
    print(f"상영시간: {time}:00")
    print(f"좌석: {seat_type} × {tickets}석")
    print("-" * 50)
    print(f"티켓 가격: {base_price:,}원 × {tickets}매")
  
    if morning_discount > 0:
        print(f"조조 할인: -{morning_discount:,}원")
  
    if is_student == "yes" and age >= 13 and age < 65:
        print(f"학생 할인 적용")
  
    print("-" * 50)
    print(f"총 결제 금액: {total_price:,}원")
    print("=" * 50)
    print("\n🍿 즐거운 관람 되세요!")
else:
    print("❌ 예매 불가!")
    print("=" * 50)
    print(f"사유: {reason}")
    print("=" * 50)
```

## 📝 **핵심 개념 정리**

`if`문은 조건이 참일 때만 코드를 실행합니다. `else`를 사용하면 조건이 거짓일 때의 동작을 지정할 수 있습니다. 여러 조건을 순차적으로 검사하려면 `elif`를 사용하며, 처음으로 참인 조건의 블록만 실행됩니다.

조건문 안에 다른 조건문을 넣는 중첩 조건문을 사용하면 더 복잡한 논리를 표현할 수 있습니다. 간단한 조건은 조건 표현식(삼항 연산자)으로 한 줄로 작성할 수 있습니다.

논리 연산자 `and`, `or`, `not`을 사용하면 여러 조건을 결합할 수 있습니다. `and`는 모든 조건이 참이어야 하고, `or`는 하나라도 참이면 되며, `not`은 조건을 반대로 만듭니다.

들여쓰기는 파이썬에서 매우 중요하며, 조건문의 블록을 구분하는 기준이 됩니다. 일반적으로 스페이스 4칸을 사용합니다.

시간대와 날씨에 따라 적절한 커피를 추천하는 프로그램을 작성하세요.

```
조건:
- 오전(6-11시): 아메리카노 추천
- 오후(12-17시): 카페라떼 추천  
- 저녁(18-23시): 디카페인 추천
- 더운 날씨(25도 이상): 아이스 추천
- 추운 날씨(15도 미만): 핫 추천
```

### **과제 2: 택시 요금 계산기**

거리와 시간에 따라 택시 요금을 계산하는 프로그램을 작성하세요.

```python
# 힌트
distance = float(input("이동 거리(km): "))
time = int(input("탑승 시간(0-23시): "))

base_fare = 4800  # 기본요금 (2km까지)
extra_fare = 0

# 2km 초과시 100원/100m 추가
if distance > 2:
    extra_fare = (distance - 2) * 10 * 100

# 심야 할증 (0-4시): 20% 추가
if time >= 0 and time < 4:
    extra_fare = extra_fare * 1.2

total_fare = base_fare + extra_fare
print(f"택시 요금: {int(total_fare):,}원")
```

## ✅ **퀴즈**

### **[초급] 1번**

다음 코드에서 출력되는 결과는?

```python
x = 10
if x > 5:
    print("A")
print("B")
```

1. A
2. B
3. A B (줄바꿈 포함)
4. 아무것도 출력 안 됨

### **[중급] 2번**

다음 중 출력되지 않는 것을 모두 고르시오.

```python
score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("D")
```

1. A
2. B
3. C
4. D

### **[중급] 3번**

다음 조건 표현식의 결과는?

```python
age = 15
result = "성인" if age >= 18 else "미성년자"
print(result)
```

1. 성인
2. 미성년자
3. True
4. False

### **[중급] 4번**

다음 코드에서 "합격"이 출력되는 조건은?

```python
score = 75
attendance = 85

if score >= 60 and attendance >= 80:
    print("합격")
else:
    print("불합격")
```

1. 점수만 60 이상
2. 출석만 80 이상
3. 둘 중 하나만 만족
4. 둘 다 만족

### **[고급] 5번**

다음 코드의 출력 결과는?

```python
x = 10
y = 20

if x > 5:
    if y > 15:
        print("A")
    else:
        print("B")
else:
    print("C")
```

1. A
2. B
3. C
4. A B

## 🔑 **퀴즈 정답 및 해설**

**1번 정답: 3**
`x > 5`는 참이므로 "A"가 출력됩니다. 그 다음 `print("B")`는 if문 밖에 있으므로 조건과 관계없이 항상 실행되어 "B"도 출력됩니다. 따라서 "A"와 "B"가 각각 한 줄씩 출력됩니다.

**2번 정답: 1, 3, 4**
`score = 85`이므로 첫 번째 조건 `score >= 90`은 거짓입니다. 두 번째 조건 `score >= 80`이 참이므로 "B"가 출력되고, 나머지 elif와 else는 실행되지 않습니다. elif-else 구조에서는 첫 번째로 참인 조건만 실행됩니다.

**3번 정답: 2**
`age = 15`이고 조건 `age >= 18`은 거짓이므로, else 뒤의 "미성년자"가 result에 할당됩니다.

**4번 정답: 4**
`and` 연산자는 두 조건이 모두 참이어야 전체가 참이 됩니다. 따라서 `score >= 60`과 `attendance >= 80`이 둘 다 만족해야 "합격"이 출력됩니다.

**5번 정답: 1**
`x > 5`는 참(10 > 5)이므로 첫 번째 if 블록으로 들어갑니다. 그 안에서 `y > 15`도 참(20 > 15)이므로 "A"가 출력됩니다. 중첩된 조건문의 경우 각 단계의 조건을 순차적으로 확인합니다.
