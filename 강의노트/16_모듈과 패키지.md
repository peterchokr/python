## 1️⃣ **모듈이란 무엇인가?**

지금까지 우리는 하나의 파이썬 파일에 모든 코드를 작성했습니다. 하지만 프로그램이 점점 커지면 어떻게 될까요? 수백, 수천 줄의 코드를 하나의 파일에서 관리하는 것은 매우 어렵습니다. 또한 다른 프로그램에서도 사용하고 싶은 유용한 함수가 있다면, 그 함수를 매번 복사해서 붙여넣어야 할까요?

이런 문제를 해결하기 위해 **모듈(Module)** 이 등장했습니다. 모듈은 관련된 함수, 변수, 클래스 등을 하나의 파이썬 파일(`.py`)에 모아둔 것입니다. 마치 도구 상자처럼, 필요한 도구(함수)들을 종류별로 정리해둔 것이라고 생각하면 됩니다.

예를 들어, 수학과 관련된 함수들은 `math` 모듈에, 날짜와 시간 관련 함수들은 `datetime` 모듈에 모아두는 것입니다.

```
모듈의 개념

┌─────────────────────┐
│   my_module.py      │  ← 하나의 파이썬 파일
├─────────────────────┤
│ def hello():        │  ← 여러 함수들을
│     print("안녕")   │     담고 있음
│                     │
│ def add(a, b):      │
│     return a + b    │
│                     │
│ PI = 3.14159        │  ← 상수도 포함
└─────────────────────┘
        ↓ import
┌─────────────────────┐
│   main.py           │  ← 다른 파일에서
├─────────────────────┤     불러와서 사용
│ import my_module    │
│                     │
│ my_module.hello()   │
│ result = my_module. │
│          add(10, 20)│
└─────────────────────┘
```

### **모듈을 사용하는 이유**

모듈을 사용하면 다음과 같은 장점이 있습니다:

**1. 코드 재사용 (Reusability)**
한 번 작성한 코드를 여러 프로그램에서 사용할 수 있습니다. 예를 들어, 계산기 함수를 만들었다면 숙제 프로그램에서도, 게임에서도, 쇼핑몰 프로그램에서도 사용할 수 있습니다. 똑같은 코드를 매번 다시 작성할 필요가 없죠!

**2. 코드 관리 (Maintainability)**
큰 프로그램을 기능별로 여러 파일로 나누면 관리가 쉬워집니다. 파일 관련 기능은 `file_utils.py`에, 계산 관련 기능은 `math_utils.py`에 넣는 식으로 정리하면, 나중에 특정 기능을 수정하거나 찾을 때 훨씬 편리합니다.

**3. 협업 (Collaboration)**
팀 프로젝트에서 각자 맡은 기능을 별도의 모듈로 만들면 동시에 작업할 수 있습니다. A는 `user_module.py`를 작성하고, B는 `product_module.py`를 작성하는 식으로 분업이 가능합니다.

**4. 이름 충돌 방지 (Namespace)**
서로 다른 모듈에 같은 이름의 함수가 있어도 문제없습니다. `module1.calculate()`와 `module2.calculate()`는 다른 함수로 취급됩니다.

## 2️⃣ **모듈 import하기**

모듈을 사용하려면 먼저 "불러오기(import)"를 해야 합니다. 마치 도서관에서 책을 빌려오듯이, 필요한 모듈을 가져와서 사용하는 것입니다.

### **방법 1: 모듈 전체 가져오기 (import)**

가장 기본적인 방법은 `import` 키워드를 사용하는 것입니다.

```python
# math 모듈 전체 가져오기
import math

# 모듈이름.함수이름() 형태로 사용
print(math.pi)           # 3.141592653589793
print(math.sqrt(16))     # 4.0 (제곱근)
print(math.factorial(5)) # 120 (팩토리얼)
```

이 방법의 특징은 모듈 안의 모든 기능을 사용할 수 있지만, 함수를 호출할 때마다 `math.`처럼 모듈 이름을 앞에 붙여야 한다는 것입니다. 이렇게 하면 "이 함수가 어느 모듈에서 온 것인지" 명확하게 알 수 있어서 코드를 읽기 쉽습니다.

### **방법 2: 필요한 것만 가져오기 (from ... import)**

모듈 전체가 아니라 필요한 함수만 골라서 가져올 수도 있습니다.

```python
# math 모듈에서 pi와 sqrt만 가져오기
from math import pi, sqrt

# 모듈 이름 없이 바로 사용 가능
print(pi)        # 3.141592653589793
print(sqrt(16))  # 4.0

# 하지만 import 하지 않은 것은 사용 불가
# print(factorial(5))  # 오류! factorial은 import 안 했음
```

이 방법은 모듈 이름을 매번 쓰지 않아도 되어 편리합니다. 하지만 코드가 길어지면 "이 함수가 어디서 왔는지" 헷갈릴 수 있으니, 자주 사용하는 몇 가지만 이렇게 가져오는 것이 좋습니다.

⚠️ **주의**: `from math import *`처럼 모든 것을 가져오는 방법도 있지만, 권장하지 않습니다. 어떤 함수들이 들어왔는지 알기 어렵고, 이름이 겹칠 수 있기 때문입니다.

### **방법 3: 별칭 사용하기 (as)**

모듈이나 함수 이름이 길 때, 짧은 별명을 붙여서 사용할 수 있습니다.

```python
# 모듈에 별칭 붙이기
import math as m  # math를 m이라는 짧은 이름으로 사용

print(m.pi)       # math.pi 대신 m.pi
print(m.sqrt(16)) # math.sqrt 대신 m.sqrt

# 함수에 별칭 붙이기
from math import factorial as fact

print(fact(5))  # 120, factorial 대신 fact 사용
```

별칭은 특히 이름이 긴 모듈을 사용할 때 유용합니다. 예를 들어 `numpy` 모듈은 거의 항상 `np`라는 별칭으로 사용합니다.

### **예제 1: 수학 계산기**

math 모듈을 활용한 과학 계산기입니다.

```python
# 과학 계산기 프로그램
import math

print("🔢" + "=" * 38 + "🔢")
print("   과학 계산기")
print("🔢" + "=" * 38 + "🔢")

while True:
    print("\n" + "=" * 40)
    print("1. 제곱근")
    print("2. 거듭제곱")
    print("3. 로그")
    print("4. 삼각함수 (sin, cos, tan)")
    print("5. 팩토리얼")
    print("6. 원의 넓이")
    print("7. 종료")
    print("=" * 40)
  
    choice = input("\n선택: ")
  
    if choice == "1":
        num = float(input("\n숫자: "))
        result = math.sqrt(num)
        print(f"√{num} = {result}")
  
    elif choice == "2":
        base = float(input("\n밑: "))
        exp = float(input("지수: "))
        result = math.pow(base, exp)
        print(f"{base}^{exp} = {result}")
  
    elif choice == "3":
        num = float(input("\n숫자: "))
        result = math.log10(num)
        print(f"log₁₀({num}) = {result}")
  
    elif choice == "4":
        degree = float(input("\n각도 (도): "))
        radian = math.radians(degree)  # 라디안으로 변환
  
        print(f"\nsin({degree}°) = {math.sin(radian):.4f}")
        print(f"cos({degree}°) = {math.cos(radian):.4f}")
        print(f"tan({degree}°) = {math.tan(radian):.4f}")
  
    elif choice == "5":
        num = int(input("\n숫자: "))
        result = math.factorial(num)
        print(f"{num}! = {result}")
  
    elif choice == "6":
        radius = float(input("\n반지름: "))
        area = math.pi * radius ** 2
        print(f"넓이 = {area:.2f}")
  
    elif choice == "7":
        print("\n계산기를 종료합니다.")
        break
  
    else:
        print("잘못된 선택입니다.")
```

## 3️⃣ **유용한 표준 라이브러리**

파이썬을 설치하면 이미 수많은 모듈이 함께 설치됩니다. 이를 "표준 라이브러리"라고 부릅니다. 별도로 설치할 필요 없이 바로 import해서 사용할 수 있는 것들입니다. 마치 스마트폰을 사면 기본으로 설치되어 있는 앱들과 같습니다.

### **random - 난수(랜덤한 숫자) 생성**

게임을 만들거나 추첨을 하거나 테스트 데이터를 만들 때 랜덤한 값이 필요합니다. `random` 모듈은 이런 난수를 생성하는 다양한 함수를 제공합니다.

```python
import random

# 1. 0.0과 1.0 사이의 랜덤한 실수
print(random.random())  # 0.523... (실행할 때마다 다름)

# 2. 특정 범위의 랜덤한 정수
dice = random.randint(1, 6)  # 주사위: 1~6 중 하나
print(f"주사위: {dice}")

# 3. 리스트에서 랜덤하게 하나 선택
menu = ["짜장면", "짬뽕", "탕수육", "볶음밥"]
today_lunch = random.choice(menu)
print(f"오늘 점심: {today_lunch}")

# 4. 리스트를 무작위로 섞기 (카드 섞기)
cards = ["♠A", "♥K", "♦Q", "♣J"]
random.shuffle(cards)  # cards 리스트 자체가 섞임
print(f"섞인 카드: {cards}")

# 5. 리스트에서 여러 개 랜덤하게 선택 (중복 없음)
numbers = list(range(1, 46))  # 1~45
lotto = random.sample(numbers, 6)  # 6개 선택
print(f"로또 번호: {sorted(lotto)}")
```

`random` 모듈은 게임, 시뮬레이션, 통계 등 다양한 곳에서 활용됩니다.

### **datetime - 날짜와 시간 다루기**

프로그램에서 현재 시간을 알아내거나, 날짜를 계산하거나, 특정 형식으로 표시할 때 사용합니다.

```python
from datetime import datetime, timedelta

# 1. 현재 날짜와 시간 가져오기
now = datetime.now()
print(now)  # 2024-02-09 14:30:25.123456

# 2. 원하는 형식으로 표시하기
print(now.strftime("%Y년 %m월 %d일"))  # 2024년 02월 09일
print(now.strftime("%H시 %M분 %S초"))  # 14시 30분 25초
print(now.strftime("%Y-%m-%d %H:%M")) # 2024-02-09 14:30

# 3. 날짜 계산하기
tomorrow = now + timedelta(days=1)      # 내일
week_later = now + timedelta(weeks=1)   # 일주일 후
yesterday = now - timedelta(days=1)     # 어제

print(f"내일: {tomorrow.strftime('%Y-%m-%d')}")
print(f"일주일 후: {week_later.strftime('%Y-%m-%d')}")

# 4. 특정 날짜 만들기
birthday = datetime(2000, 1, 1)  # 2000년 1월 1일
age_days = (now - birthday).days  # 태어난 지 며칠
print(f"태어난 지 {age_days}일")
```

`datetime`은 일정 관리, 로그 기록, D-day 계산 등에 매우 유용합니다.

**형식 코드 참고:**

- `%Y`: 4자리 연도 (2024)
- `%m`: 월 (01~12)
- `%d`: 일 (01~31)
- `%H`: 시 (00~23)
- `%M`: 분 (00~59)
- `%S`: 초 (00~59)
- `%A`: 요일 (Monday, Tuesday, ...)
- `%B`: 월 이름 (January, February, ...)

### **예제 2: 로또 번호 생성기**

random 모듈을 사용한 로또 번호 생성기입니다.

```python
# 로또 번호 생성기
import random

print("🎰" + "=" * 38 + "🎰")
print("   로또 번호 생성기")
print("🎰" + "=" * 38 + "🎰")

while True:
    print("\n생성할 번호 세트 개수 (종료: 0): ", end="")
    count = int(input())
  
    if count == 0:
        print("\n프로그램을 종료합니다.")
        break
  
    print("\n" + "=" * 40)
    print("🍀 행운의 번호")
    print("=" * 40)
  
    for i in range(count):
        # 1~45 중 6개 선택
        numbers = random.sample(range(1, 46), 6)
        numbers.sort()  # 오름차순 정렬
  
        print(f"{i+1}번째: {numbers}")
  
    print("=" * 40)
```

## 4️⃣ **자신만의 모듈 만들기**

이제 우리도 직접 모듈을 만들어 봅시다. 생각보다 간단합니다. 그냥 평소처럼 파이썬 파일(`.py`)을 만들고, 그 안에 함수를 작성하면 그게 바로 모듈입니다!

### **단계 1: 모듈 파일 만들기**

먼저 `mymath.py`라는 이름의 파일을 만들고, 다음과 같이 작성합니다:

**mymath.py:**

```python
# mymath.py
"""
나만의 수학 함수 모듈
간단한 사칙연산 함수들을 제공합니다.
"""

def add(a, b):
    """두 수를 더합니다"""
    return a + b

def subtract(a, b):
    """두 수를 뺍니다"""
    return a - b

def multiply(a, b):
    """두 수를 곱합니다"""
    return a * b

def divide(a, b):
    """두 수를 나눕니다"""
    if b == 0:
        return "0으로 나눌 수 없습니다"
    return a / b

# 모듈에 상수도 정의할 수 있습니다
PI = 3.14159
E = 2.71828

def circle_area(radius):
    """원의 넓이를 계산합니다"""
    return PI * radius * radius
```

파일 맨 위의 세 개의 따옴표(`"""`)로 둘러싸인 부분은 **독스트링(docstring)**이라고 하며, 이 모듈이 무엇을 하는지 설명하는 문서입니다. 각 함수 아래에도 독스트링을 넣어 함수의 기능을 설명할 수 있습니다.

### **단계 2: 모듈 사용하기**

이제 같은 폴더에 `main.py` 파일을 만들어서 우리가 만든 모듈을 사용해봅시다:

**main.py:**

```python
# main.py
import mymath  # 우리가 만든 모듈 불러오기

# 함수 사용
print(mymath.add(10, 20))      # 30
print(mymath.subtract(50, 20)) # 30
print(mymath.multiply(5, 6))   # 30
print(mymath.divide(100, 4))   # 25.0

# 상수 사용
print(mymath.PI)  # 3.14159

# 원의 넓이 계산
radius = 5
area = mymath.circle_area(radius)
print(f"반지름 {radius}인 원의 넓이: {area}")
```

⚠️ **주의사항**:

1. `mymath.py`와 `main.py`는 **같은 폴더**에 있어야 합니다
2. 모듈 이름에 공백이나 특수문자를 넣으면 안 됩니다
3. 모듈 이름은 파이썬 예약어(if, for 등)를 피해야 합니다

### **왜 모듈로 만드는 게 좋을까요?**

예를 들어, 계산기 프로그램, 게임, 통계 프로그램 등 여러 프로젝트를 만든다고 생각해봅시다. 이들 모두 기본적인 수학 계산이 필요합니다. `mymath` 모듈을 만들어두면:

```python
# calculator.py
import mymath
result = mymath.add(user_input1, user_input2)

# game.py
import mymath
damage = mymath.multiply(attack, multiplier)

# statistics.py
import mymath
average = mymath.divide(total, count)
```

이렇게 여러 파일에서 같은 함수를 재사용할 수 있습니다. 만약 나중에 `add` 함수를 개선하고 싶다면, `mymath.py` 파일 하나만 수정하면 됩니다. 이 모듈을 사용하는 모든 프로그램이 자동으로 개선된 함수를 사용하게 됩니다!

### **예제 3: 유틸리티 모듈**

여러 유용한 함수를 담은 모듈입니다.

**utils.py:**

```python
# utils.py
"""
유틸리티 함수 모음
"""

def print_line(char="=", length=40):
    """구분선 출력"""
    print(char * length)

def print_title(title, char="=", length=40):
    """제목 출력"""
    print(char * length)
    print(title.center(length))
    print(char * length)

def format_number(number):
    """숫자를 천 단위 콤마로 포맷"""
    return f"{number:,}"

def format_percent(value, total):
    """백분율 계산 및 포맷"""
    if total == 0:
        return "0%"
    percent = (value / total) * 100
    return f"{percent:.1f}%"

def is_valid_email(email):
    """이메일 유효성 검사 (간단)"""
    return "@" in email and "." in email
```

**main.py:**

```python
# main.py
import utils

utils.print_title("성적 통계")

scores = [85, 90, 78, 92, 88]
total = sum(scores)

print(f"\n총점: {utils.format_number(total)}점")
print(f"평균: {total/len(scores):.1f}점")

pass_count = sum(1 for s in scores if s >= 80)
print(f"80점 이상: {utils.format_percent(pass_count, len(scores))}")

utils.print_line()
```

## 5️⃣ **패키지(Package) - 모듈의 모음**

프로그램이 점점 커지면 모듈도 많아집니다. 10개, 20개, 100개의 모듈이 한 폴더에 뒤죽박죽 섞여있으면 관리하기 어렵겠죠? 이럴 때 **패키지(Package)**를 사용합니다.

패키지는 **모듈들을 담는 폴더**입니다. 마치 파일을 정리할 때 관련된 파일들을 폴더에 넣어 정리하는 것과 같습니다.

### **패키지 vs 모듈**

```
비유로 이해하기

📁 서류 정리
├── 📄 계약서.pdf          ← 파일 하나 (= 모듈)
├── 📄 영수증.pdf          ← 파일 하나 (= 모듈)
└── 📁 2024년_세금서류/    ← 폴더 (= 패키지)
    ├── 📄 1월.pdf
    ├── 📄 2월.pdf
    └── 📄 3월.pdf

💻 파이썬 코드
├── 📄 main.py             ← 메인 프로그램
├── 📄 utils.py            ← 모듈
└── 📁 mypackage/          ← 패키지
    ├── 📄 __init__.py     ← 패키지 표시 파일
    ├── 📄 math_utils.py   ← 모듈
    └── 📄 string_utils.py ← 모듈
```

### **패키지 만들기 - 실전 예제**

실제로 패키지를 만들어봅시다. 다음과 같은 구조를 만들 것입니다:

```
my_project/
│
├── main.py              ← 메인 프로그램
│
└── mytools/            ← 우리가 만들 패키지
    ├── __init__.py     ← 패키지임을 알리는 파일
    ├── calculator.py   ← 계산 관련 모듈
    └── texttools.py    ← 텍스트 처리 모듈
```

### **단계 1: 폴더 만들기**

먼저 `mytools`라는 이름의 폴더를 만듭니다.

### **단계 2: __init__.py 파일 만들기**

`mytools` 폴더 안에 `__init__.py`라는 파일을 만듭니다. 이 파일이 있으면 파이썬이 "아, 이 폴더는 패키지구나!"라고 인식합니다.

**mytools/__init__.py:**

```python
# __init__.py
"""
나만의 도구 모음 패키지
계산과 텍스트 처리 기능을 제공합니다.
"""

print("mytools 패키지가 로드되었습니다 ✓")

# 패키지 버전 정보
__version__ = "1.0.0"
__author__ = "홍길동"
```

`__init__.py`는 빈 파일이어도 되지만, 이렇게 패키지 정보를 넣거나 초기화 코드를 넣을 수 있습니다.

### **단계 3: 모듈 파일들 만들기**

**mytools/calculator.py:**

```python
# calculator.py
"""계산 관련 함수들"""

def add(a, b):
    """덧셈"""
    return a + b

def multiply(a, b):
    """곱셈"""
    return a * b

def factorial(n):
    """팩토리얼 계산"""
    if n == 0 or n == 1:
        return 1
    result = 1
    for i in range(2, n + 1):
        result = result * i
    return result
```

**mytools/texttools.py:**

```python
# texttools.py
"""텍스트 처리 함수들"""

def reverse(text):
    """문자열 뒤집기"""
    return text[::-1]

def count_words(text):
    """단어 개수 세기"""
    words = text.split()
    return len(words)

def remove_spaces(text):
    """공백 제거"""
    return text.replace(" ", "")
```

### **단계 4: 패키지 사용하기**

이제 `main.py`에서 우리가 만든 패키지를 사용해봅시다:

**main.py:**

```python
# main.py

# 방법 1: 모듈 전체 가져오기
from mytools import calculator
from mytools import texttools

print("=== 계산기 테스트 ===")
print(f"10 + 20 = {calculator.add(10, 20)}")
print(f"5! = {calculator.factorial(5)}")

print("\n=== 텍스트 도구 테스트 ===")
text = "Hello Python"
print(f"원본: {text}")
print(f"뒤집기: {texttools.reverse(text)}")
print(f"단어 수: {texttools.count_words(text)}")

# 방법 2: 필요한 함수만 가져오기
from mytools.calculator import factorial
from mytools.texttools import reverse

print(f"\n7! = {factorial(7)}")
print(f"Python 뒤집기: {reverse('Python')}")
```

실행하면:

```
mytools 패키지가 로드되었습니다 ✓
=== 계산기 테스트 ===
10 + 20 = 30
5! = 120

=== 텍스트 도구 테스트 ===
원본: Hello Python
뒤집기: nohtyP olleH
단어 수: 2

7! = 5040
Python 뒤집기: nohtyP
```

### **패키지의 장점**

1. **체계적 관리**: 관련된 모듈들을 그룹으로 묶어 정리
2. **이름 충돌 방지**: 서로 다른 패키지에 같은 이름의 모듈이 있어도 OK
3. **재사용성**: 전체 패키지를 다른 프로젝트에 복사해서 사용 가능
4. **협업 편리**: 팀원별로 다른 패키지를 만들어 작업 가능

### **더 복잡한 패키지 구조**

패키지 안에 또 다른 패키지(서브 패키지)를 만들 수도 있습니다:

```
game_project/
│
├── main.py
│
└── game/
    ├── __init__.py
    │
    ├── player/           ← 서브 패키지
    │   ├── __init__.py
    │   ├── character.py
    │   └── inventory.py
    │
    └── enemy/            ← 서브 패키지
        ├── __init__.py
        ├── monster.py
        └── boss.py
```

사용할 때:

```python
from game.player import character
from game.enemy import monster
```

이렇게 패키지를 잘 구성하면 큰 프로젝트도 깔끔하게 관리할 수 있습니다!

## 6️⃣ **외부 패키지 사용하기**

파이썬의 진정한 힘은 **외부 패키지**에 있습니다. 전 세계 개발자들이 만든 수십만 개의 패키지를 무료로 사용할 수 있습니다!

마치 스마트폰에 앱을 설치하듯이, 필요한 패키지를 설치해서 사용할 수 있습니다. 웹 크롤링, 데이터 분석, 인공지능, 게임 개발 등 거의 모든 분야의 패키지가 있습니다.

### **pip - 패키지 설치 도구**

`pip`는 파이썬 패키지를 설치하고 관리하는 프로그램입니다. 파이썬을 설치하면 자동으로 함께 설치됩니다.

**터미널(명령 프롬프트)에서 사용하는 명령어들:**

```bash
# 패키지 설치
pip install 패키지이름

# 특정 버전 설치
pip install 패키지이름==1.2.3

# 패키지 제거
pip uninstall 패키지이름

# 설치된 패키지 목록 보기
pip list

# 패키지 정보 확인
pip show 패키지이름

# 패키지 업그레이드
pip install --upgrade 패키지이름
```

### **실전 예제: requests 패키지**

`requests`는 인터넷에서 데이터를 가져올 때 사용하는 매우 유명한 패키지입니다.

**1단계: 설치하기**

```bash
pip install requests
```

**2단계: 사용하기**

```python
# 웹에서 데이터 가져오기 예제
import requests

# 웹 페이지 가져오기
response = requests.get("https://api.github.com")

# 상태 확인
print(f"상태 코드: {response.status_code}")  # 200이면 성공

# 내용 확인
if response.status_code == 200:
    print("연결 성공!")
    data = response.json()  # JSON 데이터로 변환
    print(data)
else:
    print("연결 실패")
```

### **주의사항**

1. **인터넷 연결 필요**: 패키지를 설치하려면 인터넷이 연결되어 있어야 합니다
2. **관리자 권한**: 때때로 관리자 권한이 필요할 수 있습니다

   - Windows: 명령 프롬프트를 "관리자 권한으로 실행"
   - Mac/Linux: `sudo pip install 패키지이름`
3. **가상 환경**: 프로젝트마다 다른 버전의 패키지가 필요할 때가 있습니다. 이럴 때는 "가상 환경"을 사용합니다 (나중에 배울 내용)
4. **신뢰할 수 있는 패키지만**: 누구나 패키지를 만들 수 있으므로, 유명하고 많이 사용되는 패키지를 선택하세요
