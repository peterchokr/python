# 🔑 정답 및 해설

---

## 📝 객관식 정답

---

### 🟢 초급

**문제 1. 정답: ② 13**

`x += 3`은 `x = x + 3`과 동일합니다. `x`의 초기값이 10이므로 `10 + 3 = 13`이 됩니다. 복합 대입 연산자는 현재 값에 연산을 수행한 뒤 다시 저장합니다.

---

**문제 2. 정답: ③ %**

`%`는 나머지 연산자로, 나눗셈의 나머지를 반환합니다. `/`는 일반 나눗셈, `//`는 정수 나눗셈(몫), `**`는 거듭제곱 연산자입니다.

---

**문제 3. 정답: ③ 16**

`**`는 거듭제곱 연산자입니다. `2 ** 4`는 2의 4승, 즉 `2 × 2 × 2 × 2 = 16`입니다.

---

**문제 4. 정답: ① True**

`>=`는 "크거나 같다" 비교 연산자입니다. 10은 10과 같으므로 `10 >= 10`의 결과는 `True`입니다.

---

**문제 5. 정답: ② ==**

`==`는 두 값이 같은지 비교하는 연산자이고, `=`는 값을 변수에 할당하는 대입 연산자입니다. 이 두 연산자를 혼동하지 않도록 주의해야 합니다.

---

**문제 6. 정답: ③ `*****`**

`*` 연산자를 문자열과 정수에 사용하면 문자열을 해당 횟수만큼 반복합니다. `"*" * 5`는 별 문자를 5번 반복하여 `*****`를 만듭니다.

---

**문제 7. 정답: ② False**

`not` 연산자는 불린 값을 반대로 바꿉니다. `not True`는 `False`가 되고, `not False`는 `True`가 됩니다.

---

### 🟡 중급

**문제 8. 정답: ② 13**

연산자 우선순위에 따라 곱셈이 먼저 수행됩니다. `3 * 4 = 12` → `2 + 12 - 1 = 13`. 만약 왼쪽부터 순서대로 계산했다면 19가 되겠지만, 곱셈이 덧셈/뺄셈보다 우선순위가 높습니다.

---

**문제 9. 정답: ① 25**

순서대로 계산합니다. `x = 10` → `x *= 3`이면 `x = 10 * 3 = 30` → `x -= 5`이면 `x = 30 - 5 = 25`. 복합 대입 연산자는 위에서 아래로 순서대로 적용됩니다.

---

**문제 10. 정답: ③ `age < 18 and has_ticket`**

`age = 15`이므로 `age < 18`은 `True`, `has_ticket`은 `True`입니다. `True and True = True`. ① `False and True = False`, ② `False or False = False`, ④ `not True = False`.

---

**문제 11. 정답: ② 3 2**

`17 // 5 = 3` (몫), `17 % 5 = 2` (나머지). 검증: 5 × 3 + 2 = 17. `//`는 소수점을 버리는 정수 나눗셈이고, `%`는 나머지를 구합니다.

---

**문제 12. 정답: ③ `"Hello" + 3`**

문자열과 정수를 `+` 연산자로 직접 연결할 수 없습니다. `TypeError`가 발생합니다. `"Hello" + str(3)`으로 변환해야 합니다. ④ `"Hello" * True`는 `True`가 정수 1로 취급되어 `"Hello"`가 됩니다.

---

### 🔴 고급

**문제 13. 정답: ① True**

연산자 우선순위: 비교 → not → and → or 순. `x > 3`은 `True`, `x < 10`은 `True`, `x == 1`은 `False`. → `True and True`는 `True` → `True or False`는 `True`.

---

**문제 14. 정답: ② 512**

거듭제곱 연산자(`**`)는 **오른쪽에서 왼쪽**으로 결합합니다. 따라서 `2 ** (3 ** 2)` = `2 ** 9` = `512`입니다. `(2 ** 3) ** 2`였다면 `8 ** 2 = 64`가 됩니다.

---

**문제 15. 정답: ② 3**

순서대로 계산합니다. `x = 100` → `x //= 3`이면 `x = 100 // 3 = 33` → `x %= 5`이면 `x = 33 % 5 = 3`. 검증: 33 ÷ 5 = 6 나머지 3.

**문제 25. 모범답안:**

```python
print("=== 할인 계산기 ===")

# 입력
original_price = int(input("원가: "))
discount_rate = int(input("할인율(%): "))

# 할인 계산
discount_amount = original_price * discount_rate / 100
discounted_price = original_price - discount_amount

# 배송비 결정 (5만원 이상 무료)
shipping_fee = 3000
free_shipping = discounted_price >= 50000

if free_shipping:
    final_shipping = 0
else:
    final_shipping = shipping_fee

# 최종 결제 금액
final_price = discounted_price + final_shipping

# 출력
print("--------------------")
print(f"할인 금액: {discount_amount:,.0f}원")
print(f"할인 적용가: {discounted_price:,.0f}원")
if free_shipping:
    print(f"배송비: 0원 (무료배송)")
else:
    print(f"배송비: {final_shipping:,}원")
print(f"최종 결제 금액: {final_price:,.0f}원")
```

핵심 포인트:
- 산술 연산자(`*`, `/`, `-`)로 할인 금액을 계산합니다.
- 비교 연산자(`>=`)로 무료 배송 조건을 판별합니다.
- `{:,.0f}` 포매팅으로 천 단위 구분자와 소수점 제거를 동시에 처리합니다.

### 🔴 고급

**문제 26. 모범답안:**

```python
year = int(input("연도를 입력하세요: "))

# 윤년 판별 (한 줄 조건식)
is_leap_year = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)

print(f"{year}년은 윤년입니까? {is_leap_year}")
```

핵심 포인트:
- `%` 연산자로 나누어떨어지는지 확인합니다 (나머지가 0이면 나누어떨어짐).
- `and`로 두 조건을 동시에 만족하는지, `or`로 둘 중 하나를 만족하는지 검사합니다.
- 검증: 2024 → `2024 % 4 == 0`(True) `and` `2024 % 100 != 0`(True) → True
- 검증: 1900 → `1900 % 4 == 0`(True) `and` `1900 % 100 != 0`(False) → False, `1900 % 400 == 0`(False) → 최종 False
- 검증: 2000 → 첫 번째 조건 False, `2000 % 400 == 0`(True) → 최종 True
