## 📝 객관식 정답

---

### 🟢 초급

**문제 1. 정답: ③ A (줄바꿈) B**

`x > 5`는 `True`이므로 `"A"`가 출력됩니다. `print("B")`는 if문 밖에 있으므로(들여쓰기 없음) 조건과 관계없이 항상 실행됩니다. 따라서 A와 B가 각각 한 줄씩 출력됩니다.

---

**문제 2. 정답: ② 들여쓰기 (스페이스 4칸)**

파이썬은 다른 언어와 달리 중괄호 대신 **들여쓰기**로 코드 블록을 구분합니다. 일반적으로 스페이스 4칸을 사용하며, 같은 블록의 코드는 동일한 들여쓰기를 가져야 합니다.

---

**문제 3. 정답: ② 미성년자**

`age = 15`이므로 `age >= 18`은 `False`입니다. 따라서 `else` 블록이 실행되어 `"미성년자"`가 출력됩니다.

---

**문제 4. 정답: ② else if**

`elif`는 **else if**의 줄임말입니다. 첫 번째 조건이 거짓일 때 추가로 다른 조건을 검사할 수 있게 해줍니다.

---

**문제 5. 정답: ② B**

`score = 85`이므로 첫 번째 조건 `score >= 90`은 `False`. 두 번째 조건 `score >= 80`이 `True`이므로 `"B"`가 출력됩니다. elif 구조에서는 **첫 번째로 참인 조건의 블록만** 실행되고, 나머지는 건너뜁니다.

---

**문제 6. 정답: ② 홀수**

`x = 7`이므로 `7 % 2 == 0`은 `False`입니다. 조건이 거짓이므로 `else` 뒤의 `"홀수"`가 `result`에 할당됩니다.

---

**문제 7. 정답: ④ 둘 다 만족**

`and` 연산자는 **두 조건이 모두 True**일 때만 전체가 True가 됩니다. 따라서 `score >= 60`과 `attendance >= 80`을 둘 다 만족해야 `"통과"`가 출력됩니다.

---

**문제 11. 정답: ① 성인 학생**

`age >= 18`은 `True`, `is_student`는 `True`이므로 `True and True = True`. 첫 번째 조건이 참이므로 `"성인 학생"`이 출력됩니다. elif의 `age >= 18`도 참이지만, 이미 첫 조건에서 실행되었으므로 건너뜁니다.

**문제 15. 정답: ① 삼각형 가능**

## 📝 주관식 정답

---

### 🟢 초급

**문제 16. 모범답안:**

- **`if`**: 첫 번째 조건을 검사합니다. 조건이 `True`이면 해당 블록의 코드를 실행합니다.
- **`elif`**: `if` 조건이 거짓일 때, 추가 조건을 검사합니다. 여러 개 사용 가능합니다.
- **`else`**: 위의 `if`와 모든 `elif` 조건이 거짓일 때 실행됩니다. 조건식 없이 사용합니다.

**실행 순서:** if 조건 검사 → 거짓이면 첫 번째 elif 검사 → 거짓이면 다음 elif 검사 → ... → 모두 거짓이면 else 실행. **첫 번째로 참인 조건의 블록만 실행**되고 나머지는 건너뜁니다.

---

**문제 17. 모범답안:**

```
쾌적해요
```

`temperature = 25`에서 `>= 30`은 `False`, `>= 20`은 `True`이므로 두 번째 조건의 블록이 실행됩니다. 이후 elif/else는 검사하지 않습니다.

---

**문제 18. 모범답안:**

**기본 구조:**

```python
변수 = 참일_때_값 if 조건 else 거짓일_때_값
```

**절댓값 구하기:**

```python
num = -3
abs_num = num if num >= 0 else -num
print(abs_num)  # 3
```

`num >= 0`이 `False`이므로 `-num` = `-(-3)` = `3`이 `abs_num`에 할당됩니다.
