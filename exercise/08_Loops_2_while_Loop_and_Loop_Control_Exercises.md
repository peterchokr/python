### 🟢 초급

**문제 1. 정답: ② 4번**

`count`가 0, 1, 2, 3일 때 4번 반복합니다. `count`가 4가 되면 `count < 4`가 `False`이므로 종료됩니다.

---

**문제 2. 정답: ③ 조건을 거짓으로 만드는 코드**

while 문에서 조건이 계속 True이면 무한 반복에 빠집니다. 따라서 반복 내부에서 조건을 거짓으로 만드는 코드(예: `count += 1`)가 반드시 있어야 합니다.

---

**문제 3. 정답: ② 반복문을 즉시 종료한다**

`break`는 현재 실행 중인 반복문(for 또는 while)을 즉시 종료하고, 반복문 다음의 코드로 이동합니다. 프로그램 전체를 종료하지는 않습니다.

---

**문제 4. 정답: ③ 현재 반복을 건너뛰고 다음 반복으로 넘어간다**

`continue`는 현재 반복의 나머지 코드를 건너뛰고, 반복문의 다음 반복을 시작합니다. 반복문 자체는 종료되지 않습니다.

**문제 5. 정답: ② 1 2**

`i`가 1, 2일 때 출력되고, `i`가 3이 되면 `break`로 반복이 즉시 종료됩니다. 3은 출력되지 않습니다.

**문제 6. 정답: ① 1 2 4 5**

**문제 18. 모범답안:**

**break:** 반복문을 **즉시 완전히 종료**합니다. 반복문 이후의 코드로 이동합니다.

```python
for i in range(1, 6):
    if i == 3:
        break
    print(i)  # 출력: 1 2
```

**continue:** 현재 반복의 나머지 코드를 **건너뛰고** 다음 반복을 시작합니다.

```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)  # 출력: 1 2 4 5
```

핵심 차이: `break`는 반복 자체를 끝내고, `continue`는 한 번만 건너뛰고 반복을 계속합니다.

**문제 19. 모범답안:**

실행 결과:

```

```

**동작 과정:**

| 반복 | count | total | total > 10?   |
| ---- | ----- | ----- | ------------- |
| 1    | 1     | 1     | False         |
| 2    | 2     | 3     | False         |
| 3    | 3     | 6     | False         |
| 4    | 4     | 10    | False         |
| 5    | 5     | 15    | True → break |

`while True`는 무한 반복을 만들고, 내부에서 조건을 검사하여 `break`로 탈출합니다. 이 패턴은 **"먼저 실행하고 나중에 조건 검사"**가 필요할 때 유용합니다.

| 20    | 5   | 10÷2 (짝수)   |
| 21    | 16  | 5×3+1 (홀수)  |

`steps=21`일 때 `steps > 20`이 `True`이므로 `break`. `x`가 1에 도달하기 전에 반복이 중단됩니다. 이 코드는 콜라츠 추측(어떤 자연수든 이 과정을 반복하면 결국 1에 도달한다는 추측)을 최대 20단계까지만 실행합니다.

**문제 25. 모범답안:**

```python
while True:
    print("\n=== 메뉴 ===")
    print("1. 인사")
    print("2. 계산")
    print("3. 종료")
    choice = input("선택: ")

```

- 잘못된 입력 시 `continue`로 메뉴를 다시 표시합니다.

### 🔴 고급

**문제 26. 모범답안:**

```python
start = int(input("시작 단: "))
end = int(input("끝 단: "))

# 제목 출력
for dan in range(start, end + 1):
    print(f"[{dan}단]".ljust(10), end="")
print()

# 구구단 본체 출력
for num in range(1, 10):
    for dan in range(start, end + 1):
        result = dan * num
        print(f"{dan}×{num}={result:<2}".ljust(10), end="")
    print()
```

핵심 포인트:

- 바깥 for문(`num`)이 곱하는 수(1~9)를 제어합니다.
- 안쪽 for문(`dan`)이 같은 줄에 각 단을 나란히 출력합니다.
- `ljust(10)`으로 각 열의 너비를 맞추어 세로 정렬합니다.
- `end=""`로 같은 줄에 출력하고, 안쪽 반복 종료 후 `print()`로 줄바꿈합니다.
