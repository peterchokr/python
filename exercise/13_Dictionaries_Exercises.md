### 🟢 초급

**문제 1. 정답: ② `d = {"name": "철수"}`**

딕셔너리는 **중괄호 `{}`**로 생성하고, 키와 값은 **콜론 `:`**으로 구분합니다. `=`이 아닌 `:`을 사용해야 합니다.

---

**문제 2. 정답: ② 25**

`person["나이"]`는 키 `"나이"`에 해당하는 값 `25`를 반환합니다. 딕셔너리는 키로 값에 접근합니다.

---

**문제 3. 정답: ② 3**

`menu["주스"] = 3500`으로 새 키-값 쌍이 추가되어 `{"커피": 4000, "차": 3000, "주스": 3500}`이 됩니다. 길이는 3입니다.

---

**문제 4. 정답: ② `dict.get(key)`**

`get()` 메서드는 키가 없으면 **None**(또는 지정한 기본값)을 반환하며, 오류가 발생하지 않습니다. 대괄호 `[]`는 키가 없으면 `KeyError`가 발생합니다.

---

**문제 5. 정답: ① True**

`in` 연산자는 딕셔너리의 **키**에서 존재 여부를 확인합니다. `"b"`는 키로 존재하므로 `True`입니다.

---

**문제 6. 정답: ② `{"이름": "김철수", "나이": 21}`**

기존에 있는 키에 새 값을 할당하면 **값이 수정**됩니다. 딕셔너리는 같은 키를 중복으로 가질 수 없습니다.

---

**문제 7. 정답: ② `{"x": 10, "z": 30}`**

`del d["y"]`로 키 `"y"`와 그 값이 삭제됩니다.

### 🟢 초급

**문제 16. 모범답안:**

**차이점:**

- 리스트는 **인덱스(숫자)**로 값에 접근하지만, 딕셔너리는 **키(문자열 등)**로 접근합니다.
- 리스트는 **순서가 있는** 데이터를 저장하고, 딕셔너리는 **키-값 쌍**으로 데이터를 저장합니다.
- 리스트에서 특정 값을 찾으려면 처음부터 순서대로 검색해야 하지만, 딕셔너리는 키로 **바로 찾을 수** 있어 빠릅니다.

**딕셔너리가 적합한 예시:** 전화번호부에서 이름으로 전화번호를 찾는 경우. `{"홍길동": "010-1234-5678"}` 처럼 이름(키)으로 바로 번호(값)를 찾을 수 있습니다.

---

**문제 17. 모범답안:**

```
0
85
3
```

- `get("과학", 0)` → `"과학"` 키가 없으므로 기본값 `0` 반환
- `get("국어")` → `"국어"` 키가 있으므로 값 `85` 반환
- `len(score)` → 키-값 쌍이 3개이므로 `3`

---

**문제 18. 모범답안:**

- **대괄호 `[]`**: 키가 없으면 `KeyError` **오류가 발생**합니다.
- **`get()` 메서드**: 키가 없으면 `None`을 반환하거나, 두 번째 인자로 지정한 **기본값을 반환**합니다. 오류가 발생하지 않습니다.

**실행 결과:**

```
-1
```

`d.get("c", -1)`에서 `"c"` 키가 없으므로 기본값 `-1`이 반환됩니다. 주석 처리된 `d["c"]`를 실행하면 `KeyError`가 발생합니다.

```python
text = "python is great and python is easy and python is fun"

    if word in word_count:
        word_count[word] = word_count[word] + 1
    else:
        word_count[word] = 1

# 빈도순 정렬
sorted_words = sorted(word_count.items(), key=lambda x: x[1], reverse=True)

print("=== 단어 빈도 분석 ===")
print(f"원본: {text}\n")
print("단어별 빈도 (빈도순):")
for word, count in sorted_words:
    print(f"{word}: {count}회")

print(f"\n총 단어 수: {len(words)}개")
print(f"고유 단어 수: {len(word_count)}개")
```

핵심 포인트:

- `split()`으로 문자열을 단어 리스트로 분리합니다.
- 카운팅 패턴으로 각 단어의 빈도를 셉니다.
- `sorted()`에 `key=lambda x: x[1]`을 사용하여 값(빈도) 기준으로 정렬합니다.
- `reverse=True`로 내림차순 정렬합니다.

**문제 26. 모범답안:**

```python
# 초기 재고
inventory = {
    "삼각김밥": {"가격": 1500, "수량": 20},
    "컵라면": {"가격": 1200, "수량": 30},
    "음료수": {"가격": 1800, "수량": 25}
}

total_sales = 0

def show_inventory():
    """재고 현황 출력"""
    total_value = 0
    for name, info in inventory.items():
        value = info["가격"] * info["수량"]
        total_value = total_value + value
        print(f"{name}: {info['가격']:,}원 × {info['수량']}개 = {value:,}원")
    print("-" * 29)
    print(f"총 재고 가치: {total_value:,}원")
    return total_value

# 초기 재고 출력
print("=== 편의점 재고 현황 ===")
show_inventory()

# 판매 처리
sales_list = [("삼각김밥", 3), ("음료수", 2)]

print()
for product, qty in sales_list:
    if product in inventory and inventory[product]["수량"] >= qty:
        price = inventory[product]["가격"]
        sale_amount = price * qty
        inventory[product]["수량"] = inventory[product]["수량"] - qty
        total_sales = total_sales + sale_amount
        print(f"[판매] {product} {qty}개 → {sale_amount:,}원")

# 최종 현황
print("\n=== 최종 현황 ===")
show_inventory()
print(f"총 매출: {total_sales:,}원")
```

핵심 포인트:

- 중첩 딕셔너리 `{상품명: {"가격": 값, "수량": 값}}`으로 재고를 구조화합니다.
- `inventory[상품]["가격"]`, `inventory[상품]["수량"]`으로 접근합니다.
- 판매 시 수량을 확인한 후 차감하고 매출을 누적합니다.
- 함수를 정의하여 재고 현황 출력을 재사용합니다.
