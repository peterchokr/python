## 📝 객관식 정답

---

### 🟢 초급

**문제 1. 정답: ①, ③**

`my_score`는 영문자와 밑줄로 구성되어 올바릅니다. `_count`도 밑줄로 시작하는 것이 허용되므로 올바릅니다. ②`2nd_name`은 숫자로 시작하므로 사용할 수 없고, ④`total-price`는 하이픈(`-`)이 포함되어 있어 사용할 수 없습니다.

---

**문제 2. 정답: ③ str**

큰따옴표(`""`)로 감싸진 값은 문자열(str) 타입입니다. `"홍길동"`은 텍스트 데이터이므로 str 타입으로 저장됩니다.

---

**문제 3. 정답: ②**

파이썬에서 `=`는 할당 연산자로, "오른쪽 값을 왼쪽 변수에 저장하라"는 의미입니다. 수학에서의 '같다'(등호)와는 다릅니다. 비교는 `==`를 사용합니다.

---

**문제 4. 정답: ③ True**

파이썬의 불린값은 반드시 첫 글자가 대문자여야 합니다: `True`와 `False`. ①`true`는 소문자이므로 불린값이 아니고(에러 발생), ②`FALSE`는 전부 대문자이므로 불린값이 아닙니다. ④`"True"`는 따옴표로 감싸져 있으므로 문자열(str)입니다.

---

**문제 5. 정답: ② type()**

`type()` 함수는 변수에 저장된 데이터의 타입을 반환합니다. `print()`는 값을 출력하는 함수, `int()`와 `str()`은 타입 변환 함수입니다.

---

**문제 6. 정답: ② 20**

변수는 값을 언제든 변경할 수 있습니다. `x = 10`으로 처음 10을 저장한 후, `x = 20`으로 값을 20으로 변경했으므로 `print(x)`는 20을 출력합니다.

---

**문제 7. 정답: ③ while**

`while`은 파이썬의 예약어(keyword)로, 반복문에 사용되는 단어입니다. 예약어는 변수 이름으로 사용할 수 없습니다. 나머지는 모두 올바른 변수 이름입니다.

### 🔴 고급

**문제 13. 정답: ② 3 2**

`23 // 7 = 3`은 정수 나눗셈(몫)이고, `23 % 7 = 2`는 나머지 연산입니다. 23을 7로 나누면 몫이 3, 나머지가 2입니다 (7 × 3 = 21, 23 − 21 = 2). `//`와 `%` 모두 피연산자가 정수이면 결과도 정수입니다.

### 🟡 중급

**문제 19. 모범답안:**

**오류 원인:** `input()` 함수는 항상 문자열을 반환하므로 `age`는 문자열(str) 타입입니다. 문자열과 정수(1)는 `+` 연산자로 직접 더할 수 없어 `TypeError`가 발생합니다.

**수정 코드:**

```python
age = int(input("나이를 입력하세요: "))
next_year_age = age + 1
print("내년 나이:", next_year_age)
```

`input()`으로 받은 값을 `int()`로 변환하여 정수형으로 만들면 정상적으로 덧셈이 가능합니다.

**문제 20. 모범답안:**

| 연산자 | 의미                          | `25`와 `4`의 결과 |
| ------ | ----------------------------- | --------------------- |
| `/`  | 일반 나눗셈 (항상 float 반환) | `25 / 4 = 6.25`     |
| `//` | 정수 나눗셈 (몫만 반환)       | `25 // 4 = 6`       |
| `%`  | 나머지 연산                   | `25 % 4 = 1`        |

검증: 25 = 4 × **6** + **1** (몫 = 6, 나머지 = 1)

**단계별 설명:**

6. `str(7)` → 정수 7을 문자열 "7"로 변환 (int → str)
7. `"7" + a` → 문자열 연결: `"7" + "3" = "73"` → `d = "73"` (str)

```python
print(f"{d} → {type(d).__name__}")
```

핵심: `type(변수).__name__`을 사용하면 `<class 'int'>` 대신 `int`만 깔끔하게 출력할 수 있습니다. 4가지 기본 데이터 타입(int, float, str, bool)을 확인하는 연습입니다.

**문제 25. 모범답안:**

```python
print("=== 성적 계산기 ===")

# 점수 입력 (문자열 → 정수 변환)
korean = int(input("국어 점수: "))

total = korean + english + math
```

핵심 포인트:

- `/` 연산자로 나누면 자동으로 float 타입이 되어 소수점 평균이 계산됩니다.
- `:.1f`는 소수점 첫째 자리까지 표시하는 서식 지정입니다.

### 🔴 고급

**문제 26. 모범답안:**

```python
# 초 입력
total_seconds = int(input("초를 입력하세요: "))

# 시, 분, 초 계산
hours = total_seconds // 3600        # 1시간 = 3600초
remaining = total_seconds % 3600     # 시간 계산 후 남은 초
minutes = remaining // 60            # 1분 = 60초
seconds = remaining % 60             # 분 계산 후 남은 초

# 결과 출력
print(f"{total_seconds}초 = {hours}시간 {minutes}분 {seconds}초")
```

핵심 포인트:

- `//` (정수 나눗셈)으로 몫을 구하고, `%` (나머지 연산)로 남은 값을 구합니다.
- 3725초 → 3725 // 3600 = **1시간**, 3725 % 3600 = 125초 남음 → 125 // 60 = **2분**, 125 % 60 = **5초**
- `//`와 `%`를 조합하면 단위 변환 문제를 효과적으로 풀 수 있습니다.
